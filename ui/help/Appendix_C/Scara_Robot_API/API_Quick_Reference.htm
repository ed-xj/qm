API Quick Reference

Abort – Abort robot motion quickly and de-servo the motors

Type: immediate

abort

 

Align – Start an alignment operation on the PA-800 prealigner

Type: queued

align {1 | 2} {notch | flat} {wafer-size} [station name]

 

Config – Access system configuration settings

Type: queued

config {save|restore} {filename}

config get (category (item))

config set {category} {item} (=) {value}

 

Delete – Delete one, or all stations

Type: queued

delete {station-name | *}

 

DeleteFrame – Delete one, or all user-defined frames

Type: queued

DeleteFrame {frame-name | *}

 

Dir – List a directory of station database files in non-volatile memory

Type: immediate

dir (match-pattern)

 

EFnum – Change the current end-effector number

Type: queued

efnum {1 | 2}

 

Event – Send a message ("user event") from the controller to the host

Type: queued

 

Fiduciary – Set the desired angle of the aligned fiducial notch or flat

Type: queued

fiduciary {angle}

 

Flip – Rotate the end-effector flipper to a specified angle

Type: queued

flip {abs|rel}{angle1, angle2...} {speed|fspeed} {1-100} {acc|facc} {1-100}

 

Flipper – Command the end-effector flipper

Type: queued

flipper {on | off | center | zero | home}

 

Flush – Abort motion smoothly and flush the command queue

Type: Immediate

flush

 

Get – Return the state of an end-effector or input/output line

Type: Immediate

get grip {1|2}

get {input|output} {1-8}

 

GetFrame(s) – Output listing of user-defined frames

Type: immediate

getframe(s) (frame-name)

 

GetStation(s) – Output listing of station data

Type: immediate

getstation(s) (station-name)

 

GetXYZWPR – Return station position data in world coordinates

Type: Immediate

getxyzwpr {station} (frame {userframe})

 

Grip – Open or close the gripper (or vacuum) on a specified end-effector

Type: queued

grip {1 | 2} {on | off}

 

Help – Provide a list of commands and syntax

Type: immediate

help (command)

 

Init – Initialize the specified device

Type: queued

init {flipper | prealigner} {comm port} {baud rate}

 

LCM – Last Completed Motion; report last completed motion

Type: immediate

lcm

 

Learn – Set a station pose to the current position and end-effector number

Type: queued

learn {station} {inside | perch} (index)

 

Logfile – Output a record of commands, responses, and events

Type: immediate

logfile (old) (number)

 

Map – Perform a wafer-mapping operation on a station

Type: queued

map {station} (data) (motion-arguments)

 

Mapper – Operate the wafer mapper

Type: queued

mapper {init | on | off | status}

mapper data [XYZWPR]

mapper data frame (frame-name)

 

Move – Move to a specified station pose (inside or perch)

Type: queued

move {station} (motion-arguments)

 

Output – Set the state of an output line

Type: queued

output {1-8} {off | on}

 

Passcontrol – Relinquish control to the teach pendant

Type: immediate

passcontrol

 

Pause – Suspend robot motion for resumption later

Type: immediate

pause

 

Pick – Move to station, pick up wafer, and clear station

Type: queued

pick {station} (motion-arguments)

 

Place – Move to station, release wafer, and clear station

Type: queued

place {station} (motion-arguments)

 

RCP – Read Current Position; output current robot position

Type: Immediate

rcp

rcp xyzwpr

rcp xyzwpr frame (frame-name)

 

Recover – Move to the perch position of the current station

Type: queued

recover [speed]

 

Release – Do the stroke and ungrip required to release a wafer

Type: queued

release {station} (motion-arguments)

 

Remove – Delete a station database from non-volatile memory

Type: immediate

remove {file-name | *}

 

Reset – Abort motion smoothly, flush command queue and clear faults

Type: Immediate

reset

 

Restore – Read stored station, frame data into current database

Type: queued

restore {file-name}

 

Resume – Resume motion after Pause command

Type: immediate

resume

 

Retract – Move safely out from inside a station to perch position

Type: queued

retract (({acc|blend|linear|speed} {num}) …)

 

RTP – Read Target Position; return the commanded (target) position

Type: immediate

rtp

 

Save – Saves current station data, user-defined frames

Type: queued

save {file-name}

 

Seize – Do the stroke and grip required to seize a wafer

Type: queued

seize {station} (motion-arguments)

 

Set – Set a station database attribute

Type: queued

set {station} {inside | perch} {Z | *} {{shoulder | *} {elbow | *} {wrist | *} {flip | *}

set ( station) {inside | perch} near {station2} xyzwpr {x} {y} { z} {w} {p} {r}

set {station} {pitch | stroke | slots | efnum} {num}

set {stations} type {cassette | prealigner [{1 | 2}]| elevator}

set rtheta {true | false}

set (station) mapcal {LeftLow, LeftHigh, RightHigh, RightLow}

set (station) {inside | perch} near ( station2) frame (framename) xyzwpr {x} {y} {z} {w} {p} {r}

set {station} inputs {approach} {inside} {wafer}

 

SetFrame – Set a user frame definition

Type: queued

setframe {frame} {x} {y} {z} {w}

 

Sim – Simulate a function or hardware

Type: immediate

sim {flipper | gripper | gripdetect | prealigner [{1 | 2 | *}]| robot | vacdetect | vacuum}

sim waitinput {*|1|2|3|4|5|6|7|8}

 

Speed – Set speed-override setting

Type: queued

speed (percent)

 

Status – Report various status information

Type: immediate

status

 

Unsim – Un-simulate a function or hardware

Type: immediate

unsim {flipper | gripper | gripdetect | prealigner [{1 | 2 | *}]| robot | vacuum | vacdetect}

unsim waitinput {*|1|2|3|4|5|6|7|8}

 

VacOff – Turn off end-effector vacuum

Type: queued

vacoff (end-effector-number)

 

VacOn – Turn on vacuum

Type: queued

vacon (end-effector-number)

 

VacThreshSet – Set vacuum threshold values

Type: queued

vacthreshset {house-atmos} {ef1-diff} {ef2-diff}

 

Version – Display version numbers

Type: immediate

version

 

Wait – Wait for the completion of an action

Type: queued

wait {align | modone | flipper | prealigner [{1 | 2}]}

wait input {1 | 2 | 3 | 4 | 5 | 6 | 7 | 8} {off | on}
