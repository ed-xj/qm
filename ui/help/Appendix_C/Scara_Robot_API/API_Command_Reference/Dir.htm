Dir - List a directory of station database files in non-volatile memory.

type: immediate

Syntax

dir (match-pattern)

Arguments

match-pattern: An optional filename wildcard. This is not a strict "regular expression", but a DOS-type filename pattern where " * " matches any sequence of characters, and "?" any single character.

Description

Lists the directory of station database files. Each listing is printed on a separate line containing the following information: file size in bytes, date last saved, and file name.

 

See also: Remove, Restore, Save.

Events

!E END-DIR<tab>{host command, as received}

Examples

dir

>

!S 211 jun 13 2001 alpha

!S 708 jun 12 2001 station1

!S 3946 may 29 2001 stations

!E END-DIR dir

 

dir stat*

>

!S 708 jun 12 2001 station1

!S 3946 may 29 2001 stations

!E END-DIR dir stat*
