Flush - Abort motion smoothly and flush the command queue.

type: immediate

Syntax

flush

Arguments

None.

Description

This command is currently identical in effect to the RESET command.

 

FLUSH flushes all pending motions and attempts to clear error conditions and to prepare the robot to perform motions.

 

If the robot is in motion when the command is issued, it will come to a smooth stop.

 

Note: when the controller issues an error response or an error event, it will not respond to most commands until it receives a reset command.

 

See also: Reset, Abort, Pause.
