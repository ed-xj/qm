Seize - Do the stroke and grip required to seize a wafer.

type: queued

Syntax

seize {station} (motion-arguments)

Arguments

See Appendix A, MOTION COMMAND ARGUMENTS.

Description

Strokes up and grips a wafer, using vacuum or the gripper, as is appropriate. The end-effector number is an attribute of the station. Whether the end-effector has vacuum or a gripper is defined in a robot configuration file by IRSI at the factory.

 

See also: Pick.

Events

Note: For clarity, only the initial parts of the event messages are given below. Each one will be terminated, as always, with a tab character followed by the host’s command that precipitated the event.

 

!E PA-OFF-SEIZE - (Prealigner station only) - Prealigner has released wafer, and is ready for pickup

!E GRIP-RETRY-SEIZE –(Gripping EE Only) – The first grip attempt failed.

!E GRIP-SEIZE - (Gripping EE only) – The wafer has been gripped.

!E STROKE-SEIZE – The robot reached its stroked-up position.

!E VACON-SEIZE - (Vacuum EE only) – Vacuum sensed, wafer present.

!E END-SEIZE – The seize operation has completed.

Examples

SIEZE PDOONE INDEX -1
