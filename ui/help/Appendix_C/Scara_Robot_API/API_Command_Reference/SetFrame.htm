SetFrame - Set a user frame definition.

type: queued

Syntax

setframe {frame} {x} {y} {z} {w}

Arguments

frame: A contiguous string of alpha-numeric characters, or any printable text enclosed in double quotation marks.

 

x, y, z, w: (in millimeters) and the W (Yaw) rotation (in degrees) of a user-defined frame. Parameters are evaluated in world frame.

Description

You may define up to nine frames of reference ("user frames"). Because the end-effector is constrained to lie in a plane parallel to the world frame's xy plane, we do not support frames with an arbitrary pitch. Thus the frame is defined only by its x,y,z origin in the world frame, and its w rotation (yaw) relative to the world frame. Frames may be specified in the API calls SET, RCP, and MAPPER DATA. Frames may also be used for teach pendant jogging.

 

See also, SAVE, RESTORE, GETFRAMES, SET, RCP, and MAPPER.

Events

!E END-SETFRAME<tab>{host command, as received}

Examples

Setframe "my frame" 23.692, -66.392, 88.02, 90
