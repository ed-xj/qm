RCP - Read Current Position; output current robot position.

type: immediate

Syntax

rcp

rcp xyzwpr

rcp xyzwpr frame (frame-name)

Arguments

The optional argument xyzwpr will cause the controller to list the position in XYZWPR format. If the frame is given in the argument list, the xyzwpr coordinates sent to the host will be relative to the named user-defined coordinate frame.

Description

Sends the host the position of the robot, in joint angle space. The first status lines returned will indicate the actual position of the robot. If there are stations defined in the station database, the remaining lines will first show the closest station and pose, (inside or perch), and index, followed by the joint angles of the closest pose and index. The controller will return Z, shoulder, elbow, and wrist angles. If the station is for an end-effector that has a flipper, it will also return the flipper angle.

 

See also: RTP.

Response

[Robot status lines]...

Events

!E END-RCP<tab>{host command, as received}

523 rcp

523>

!S 523 rcp ACTUAL-S 42.432

!S 523 rcp ACTUAL-E -32.902

!S 523 rcp ACTUAL-W 192.703
