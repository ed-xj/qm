Retract - Move safely out from inside a station to perch position.

type: queued

Syntax

retract (({acc|blend|linear|speed} {num}) …)

Arguments

Optionally, the following subset of motion command arguments may be added to the retract command: blend, acc, speed, linear.

 

See Appendix A, MOTION COMMAND ARGUMENTS.

Description

This command simplifies a common task: pull the robot arm out of a station to that station’s perch position. It is identical to issuing a move command, but without the necessity to specify the station name, perch or inside (retract always moves to perch position of the station in question), slot number, etc. The station implied when the retract command is issued is that in which the robot will find itself when the queue of motion commands had been completed.
