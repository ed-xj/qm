Save - Saves current station data, user-defined frames.

type: queued

Syntax

save {file-name}

Arguments

Specify a file-name where the current station database, frame definitions are to be saved. The file name must eight characters or less of standard DOS filename characters.

Description

Saves the current station data and user-defined frames to non-volatile memory. Each time a file is saved, memory is queried for available free bytes and, if necessary, purged beforehand. The purge operation may take up to five minutes, during which an event message will inform the user, "!S Purging FR: drive. This may take a few minutes. Please wait."

 

See also: Getstations, Restore, Remove, Dir, Delete, SetFrame, DeleteFrame.
