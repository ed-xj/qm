VacOff - Turn off end-effector vacuum.

type: queued

Syntax

vacoff (end-effector-number)

Arguments

end-effector-number: Defaults to 1 if not specified.

Description

Turns off vacuum to specified end-effector regardless of wafer presence or position, as opposed to GRIP.

Events
