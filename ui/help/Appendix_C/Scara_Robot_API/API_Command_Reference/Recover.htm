Recover - Move to the perch position of the current station.

type: queued

Syntax

recover [speed]

Arguments

speed: Defaults to 10 if not specified.

Description

A host can use this command if the robot has discontinued (for any reason excluding a power-cycle) a pick or place process. If the robot is gripping a wafer while this command is issued, the robot will return to the perch position gripping the wafer. If the robot is not gripping a wafer while this command is issued, the robot will return to the perch position with out a wafer.
