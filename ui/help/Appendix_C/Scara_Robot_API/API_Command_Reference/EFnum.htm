EFnum - Change the current end-effector number.

type: queued

Syntax

efnum {1 | 2}

Arguments

The number of end-effector to be selected (1 or 2).

Description

Sets the current end-effector number that the robot uses for jogging and learning stations. The current end-effector number does not affect motion commands, as they are based on an end-effector number stored in the station data. When the host sends a learn command, the learned station will be set to use the end-effector number that is current. When the system comes up, the current end-effector number is 1 (default).

Events

!E END-EFNUM<tab>{host command, as received}
