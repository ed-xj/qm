Fiduciary - Set the desired angle of the aligned fiducial notch or flat.

type: queued

Syntax

fiduciary {angle}

Arguments

The one argument is a fiducial angle, measured in degrees.

Description

The one argument is the angle where you want the fiducial notch or flat of the wafer to be after it is prealigned and picked up. The angle is reckoned relative to the line on the prealigner extending from the center of rotation to the light sensors. Looking down on the prealigner, a positive value corresponds to counterclockwise rotation of the wafer.

 

See also: Align.
