Sim - Simulate a function or hardware.

type: immediate

Syntax

sim {flipper | gripper | gripdetect | prealigner [{1 | 2 | *}]| robot | vacdetect | vacuum}

sim waitinput {*|1|2|3|4|5|6|7|8}

Arguments

flipper: Simulates the presence of a flipper/gripper assembly, including Gripper.

 

gripper: Simulates the presence of a gripper and sensor, including Gripdetect.

 

gripdetect: Simulates wafer presence during a grip.

 

prealigner: Simulates the presence of the prealigner specified by the last "align" command. If using the optional arguments "1" or "2", simulates the specified prealigner. Using the optional argument " * " simulates all prealigners.

 

robot: Simulates the presence of a complete M800 robot, including end-effectors and prealigners.

 

vacdetect: Simulates vacuum detection.

 

vacuum: Simulates the presence of vacuum, including Vacdetect.

 

waitinput: Simulates an awaited state on a digital input or all inputs (splat " * ") when set. Example: a sensor connected to Input 1 is used to indicate the presence of a wafer at a given position, which presence will prevent further robot motion. If WaitInput 1 is simmed, motion will be allowed to continue regardless of the Input 1 state.

Description

None.

Events

!E END-SIM<tab>{host command, as received}
