Logfile - Output a record of commands, responses, and events.

type: immediate

Syntax

logfile (old) (number)

Arguments

The optional keyword old means to return data from the previous session. A session begins when the controller is restarted.

 

The optional number, a positive integer, tells how many lines to return.

Description

Sends the host a record of commands, responses, and events, either from the current session, or the previous session. A session begins when the controller is restarted.

 

All logfile lines begin with the prefix LOG followed by a space. Commands from the host are printed inside square braces.

Response

{log file lines}...

Events

!E END-LOGFILE<tab>{host command, as received}

Examples

125 logfile 6

125>

LOG [123 restore newstations]

LOG 123 >

LOG !E END-RESTORE123 restore newstations

LOG [124 speed 10]

LOG 124>

LOG !E END-SPEEDspeed 10

!E END-LOGFILE 596 logfile 6
