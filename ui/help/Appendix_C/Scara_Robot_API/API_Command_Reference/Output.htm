Output - Set the state of an output line.

type: queued

Syntax

output {1-8} {off | on}

Arguments

Output-number: The number of the output line to set. Valid output line numbers are 1 to 8 in the base system, optional I/O may increase the available lines.

 

off: The specified line is taken to signal ground.

on: The specified line goes into a high-impedance (open-collector) state.

Description

Output sets the state of a digital output line. Output lines are open-collector circuits with a capability of handling up to 30 volts. +24V and signal ground is supplied on the I/O connector for use with these lines.

Events

!E END-OUTPUT<tab>{host command, as received}

Examples

move "via 1" ; move robot clear

wait modone; wait until it's there

output 1 on; close the door

 

The robot will move to the "via 1" station and decelerate to a stop. As soon as the robot has stopped, output 1 will be set to 1. Without the WAIT MODONE command, output 1 would be set on as soon as the move to "via 1" had started.
