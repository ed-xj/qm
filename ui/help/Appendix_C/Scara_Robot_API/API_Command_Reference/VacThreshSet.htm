VacThreshSet - Set vacuum threshold values.

type: queued

Syntax

vacthreshset {house-atmos} {ef1-diff} {ef2-diff}

Arguments

house-atmos: Maximum allowed house vacuum pressure as a percentage of atmosphere.

 

ef1-diff: Maximum allowed percentage difference (of one atmosphere) between end-effector 1and house vacuum.

 

ef2-diff: Maximum allowed percentage difference (of one atmosphere) between end-effector 2 and house vacuum.

 

Must be specified even if the robot has only one end-effector.

Description

House and end-effector vacuums are measured using analog sensors. This command specifies the level of house vacuum that must exist for the robot to function, and also how good the vacuum level must be (relative to house) before considering a wafer "gripped". By using analog measurements and relative comparisons, a very reliable method of detecting wafer presence results that tracks with fluctuations in house vacuum.

 

For example, a house-atmos setting of 40 means that the house vacuum must be 40% or less of atmospheric pressure; a smaller number indicating greater vacuum. An end-effector setting of 20 would mean that in order for a wafer to be considered properly gripped, the end-effector pressure must be no more than 20% of house vacuum; a smaller number means there must be a better seal, or grip, of the wafer. Typical settings would be 40 or 50 for house vacuum, and 20 to 25 for the end-effectors.

 

These settings are only good until power-down unless a CONFIG SAVE command is executed after VACTHRESHSET.

 

Current settings can be checked with the STATUS command.

Events

!E END-VACTHRESHSET<tab>{host command, as received}

Examples

vacthreshset 40 20 20

>

!E END-VACTHRESHSET<tab>vacthresset 40 20 20
