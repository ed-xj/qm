Get - Return the state of an end-effector or input/output line.

type: immediate

Syntax

get grip {1|2}

get {input|output} {1-8}

Arguments

grip: Report whether an end-effector is holding a wafer. Default is end-effector one.

input: Report the status of a digital input line.

output: Report the state of a digital output line.

Description

GET: Returns the state of its argument. In the case of GET GRIP, if the end-effector has correctly gripped a wafer, ON is returned, else OFF. For a gripping end-effector, this is independent of any previous commanded state; if a "GRIP 1 OFF" command has failed, "GET GRIP 1" could be used to determine if the wafer is still securely held.

 

GET INPUT: Returns On if the digital input line is "high" (above 3 volts), else Off.

 

GET OUTPUT: Returns the actual state of a digital output, regardless of previous commanded state; On for "high" (above 3 volts) else Off. For example, if for some reason the digital output line is being held low by an external sink, 0 would be returned regardless of the last OUTPUT command.

 

See also: GRIP, OUTPUT.

Events

!S {argument} {value}

!E END-GET<tab>{host command, as received}

Examples

GET GRIP 1

>

!S GRIP ON

!E END-GETGET GRIP 1

 

GET INPUT 3

>

!S INPUT-3 Off

!E END-GETGET INPUT 3

 

GET OUTPUT 2

>

!S OUTPUT-2 Off

!E END-GETGET OUTPUT 2
