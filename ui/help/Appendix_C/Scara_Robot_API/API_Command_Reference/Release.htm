Release - Do the stroke and ungrip required to release a wafer.

type: queued

Syntax

release {station} (motion-arguments)

Arguments

See Appendix A, MOTION COMMAND ARGUMENTS.

Description

Strokes down and ungrips a wafer, turning off the vacuum or the gripper as appropriate. Stroking down means the end-effector moves to the low position for the index. See Pick or Place for a definition of the low position.

Events

Note: For clarity, only the initial parts of the event messages are given below. Each one will be terminated, as always, with a tab character followed by the host’s command that precipitated the event.

 

!E VACOFF-REL - (Vacuum EE Only) – Vacuum was turned off.

!E GRIP-RETRY-REL (Gripping EE Only) – The first ungrip attempt failed.

!E UNGRIP-REL - (Gripping EE only) – The wafer has been released.

!E STROKE-REL – The robot reached its stroked-down position.

!E PA-ON-REL - (Prealigner station only) – Prealigner has captured wafer. Robot may exit.

!E END-REL – The release operation has completed.
