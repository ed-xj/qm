Remove - Delete a station database from non-volatile memory.

type: immediate

Syntax

remove {file-name | *}

Arguments

Specify a file-name to be deleted. The file name must be eight characters or less, of standard DOS filename characters.

 

An asterisk (or "splat," *) passed as an argument will delete ALL station databases.

Description

Deletes the specified station database from non-volatile memory.

 

See also: Getstations, Restore, Remove, Dir.

Events

!E END-REMOVE<tab>{host command, as received}
