Passcontrol - Relinquish control to the teach pendant.

type: immediate

Syntax

passcontrol

Arguments

None.

Description

Relinquishes host control of the controller. Control is passed to the next entity to request it. If the teach pendant is subsequently used to jog the robot, the teach pendant assumes control. If the host issues a move command after control is passed, but before action is taken on the teach pendant, then the host regains control.

Events
