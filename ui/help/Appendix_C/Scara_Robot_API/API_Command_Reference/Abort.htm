Abort - Abort robot motion quickly and de-servo the motors.

type: immediate

Syntax

abort

Arguments

None.

Description

This commands stops the robot abruptly, de-servos the motors, and applies the Z-axis brake. All queued commands are flushed. This command is used when de-servoing the motors and applying the Z-axis brake is desired.

 

Note: This command can be used to stop a moving robot, but the "Reset" command is referred in non-emergency situations. For emergency situations, either the teach pendant or system e-stop button should be used.

 

WARNING: This command should be used before turning off the controller to allow the Z-brake to set. Failure to do so may result in 2-3mm of Z-axis drop on power-off, possibly resulting in payload or equipment damage.

 

See also: Reset.

Events

!E ERROR 67 - CTR Controller event message: SRVO-031
