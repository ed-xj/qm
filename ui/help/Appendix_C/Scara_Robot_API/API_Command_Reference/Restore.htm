Restore - Read stored station, frame data into current database.

type: queued

Syntax

restore {file-name}

Arguments

Specify a file-name where a station database, user frames were previously saved. The file name must eight characters or less, and of standard DOS naming convention characters. No file extension is allowed.

Description

Restores saved station data, user-defined frames from non-volatile memory into the current station data. Does NOT delete current station data (though identical station names will be overwritten). Use "delete *", "deleteFrames *"to clear all station data before calling "restore," if desired.

 

See also: Save, Delete, DeleteFrame, SetFrame.

Events

!E END-RESTORE<tab>{host command, as received}

Examples

123rstr restore STAT ; Valid restore command

124rstr restore TOOLONGANAME ; ERROR! File name > 8 chars
