Delete - Delete one, or all stations.

type: queued

Syntax

delete {station-name | *}

Arguments

An asterisk ("star" or "splat") " * " tells the controller to empty the station database. It does not empty the user-defined frames. Use DeleteFrame.

 

If the argument is a station-name, it the controller will delete the station named by that literal.

Description

Deletes a station from the working station database. If the argument is an asterisk, it deletes the whole database.

 

Note: The delete command only alters the current working station database. It does not modify any station database that might have been saved previously with the save command. Station names may include blanks or other whitespace, in which case the host must send the name enclosed in double quotation marks. Station names are not case sensitive.

 

See also: Save, Restore, Remove.

Events

!E END-DELETE<tab>{host command, as received}

Examples

4231 delete * ; Delete the whole station database

29del delete Via1 ; Delete the station named "Via1"

delete "Lp_one" ; Delete the station named "Lp_one"

delete "LP_ONE" ; Equivalent to the above
