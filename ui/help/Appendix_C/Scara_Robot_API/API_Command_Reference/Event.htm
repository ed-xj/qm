Event - Send a message ("user event") from the controller to the host.

type: queued

Syntax

event {literal}

Arguments

If present, the argument is a literal chosen by the host.

Description

The one argument is a literal string chosen by the host. When the controller reaches the event command in the command queue, it will send a message to the host with the prefix !U.
