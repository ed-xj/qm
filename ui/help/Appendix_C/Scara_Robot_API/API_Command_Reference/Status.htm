Status - Report various status information.

type: immediate

Syntax

status

Arguments

None.

Description

Reports various information relating to the status of the robot. The information is returned by keyword and value.

 

Each keyword/value pair is prefixed with !S followed by the cookie, if any, followed by the word status.

 

The list of values that the status command currently returns is not complete. Programmers should code the host program to ignore unrecognized keywords.

Response

!S[cookie] status [status keyword/value pair]...

Events

!E END-STATUS<tab>{host command, as received}

Examples

456 status

456>

!S 456 STATUS Paused = false

!S 456 STATUS Fault = true

< et cetera >

!E END-STATUS 456 status
