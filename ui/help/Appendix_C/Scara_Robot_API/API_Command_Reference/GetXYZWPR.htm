GetXYZWPR - Return station position data in world coordinates.

type: immediate

Syntax

getxyzwpr {station} (frame {userframe})

Arguments

station: Any station name in the current station database. Optional. Syntax is identical to GETSTATION.

userframe: A user-defined coordinate frame to be used as the frame of reference for the returned data. By default the robot "world" coordinate frame is used.

Description

GETXYZWPR returns the world coordinates (optionally, within a defined userframe) of the PERCH and INSIDE positions of the specified station (or for all stations within the current station database if no station is specified). Six floating-point values are returned in the following order for each position: X, Y and Z denote their respective axis offsets. W stands for yaw (rotation around the Z axis), P for pitch (rotation around the Y axis), and R for roll (rotation around the X axis).

 

Offsets are reported in millimeters, and rotations in degrees; positive degrees are counter-clockwise when looking along the positive line of the axis to the origin.

 

See also: GetFrame, rcp, SetFrame.

Events

!S "station" inside {X} {Y} {Z} {W} {P} {R}

!S "station" perch {X} {Y} {Z} {W} {P} {R}

!E END-GETXYZWPRgetxyzwpr

Examples

getxyzwpr foo

>

!S "foo" inside 370.564, 185.695, -2.935, -5.548, 0.000, 0.000

!S "foo" perch 370.564, 185.695, -2.935, -5.548, 0.000, 0.000

!E END-GETXYZWPRgetxyzwpr foo
