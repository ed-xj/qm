Pick - Move to station , pick up wafer, and clear station.

type: queued

Syntax

pick {station} (motion-arguments)

Arguments

See MOTION COMMAND ARGUMENTS, Appendix A.

Description

Moves to the perch position of the indicated station, then moves inside, picks up a wafer, and moves out to the perch position.

 

The action of picking up the wafer has a number of subactions that can vary depending upon what kind of end-effector is in use, and upon whether the station is a prealigner or not. A typical set of actions would be to come into the station low, stroke up and grip the wafer, then exit from the station high holding the wafer.

 

For a given pose (inside or perch) and index, the low Z elevation is defined by:

Low(index).Z = pose.Z + (index-1) * station.pitch – station.stroke

Similarly, the high Z elevation is defined by:

High(index).Z = pose.Z + (index-1) * station.pitch

 

Notice that the high Z elevation defined for a pose is the "stroked up" position. To set station.pitch, and station.stroke, see the Set command. To set pose.Z (along with the shoulder, elbow, wrist, and optional flipper positions), see Set and Learn.

Events

Note: For clarity, only the initial parts of the event messages are given below. Each one will be terminated, as always, with a tab character followed by the host’s command that precipitated the event.

 

!E PERCH-PICK – The robot has reached the lower perch position.

!E INSIDE-PICK – The robot reached the inside of the station.

!E PA-OFF-PICK - (Prealigner station only)Prealigner has released wafer, and is ready for pickup.

!E GRIP-RETRY-PICK –(Gripping EE Only) – The first grip attempt failed.

!E GRIP-PICK - (Gripping EE Only) – The wafer has been gripped.

!E STROKE-PICK – The robot has stroked up.

!E VACON-PICK - (Vacuum EE only) – Vacuum sensed; wafer present.

!E END-PICK – The robot has reached the upper perch position. The pick command is finished.
