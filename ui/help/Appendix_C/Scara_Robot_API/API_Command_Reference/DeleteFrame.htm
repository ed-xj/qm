DeleteFrame - delete one, or all user-defined frames

type: queued

Syntax

DeleteFrame {frame-name | *}

Arguments

An asterisk ("star" or "splat") " * " tells the controller to empty the frame database in volatile memory. If the argument is a frame-name, the controller will delete the frame named by that literal.

Description

Deletes a user-defined frame from the working frame database. If the argument is an asterisk, it deletes the whole database.

 

Note: The DeleteFrame command only alters the current working frame database. It does not modify any station and frame database that might have been saved previously with the save command. Frame names may include blanks or other whitespace, in which case the host must send the name enclosed in double quotation marks. Frame names are not case sensitive.

 

See also: Save, Restore, SetFrame.

Events

!E END-DELETEFRAME<tab>{host command, as received}

Examples

deleteframe * ; Delete all frames

deleteframe Frame1 ; Delete frame "Frame1"
