Init - Initialize the specified device.

type: queued

Syntax

init {flipper | prealigner} {comm port} {baud rate}

Arguments

Output-number: the number of the output line to set. Valid output line numbers are 1 to 8 in the base system; optional I/O may increase the available lines.

flipper: initialize the flipper.

preligner: initialize the prealigner.

comm port: specify the comm port the device is attached to. Syntax is "comx", where x is the comm port number.

baud rate: specify the baud rate for the device. Typical baud rates are 9600, 19200, and 57600.

Description

Init is used to initialize certain optional devices attached to the robot controller. Currently, only flippers and prealigners fall into this category. For the M820C controller, prealigner 1 is on comm port 9, and prealigner 2 is on comm port 10. The flipper is on comm port 12.

Events

!E END-INIT<tab>{host command, as received}

Examples

init prealigner com9 19200";

init flipper com12 19200
