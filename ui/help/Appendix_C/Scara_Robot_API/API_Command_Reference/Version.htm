Version - Display version numbers.

type: immediate

Syntax

version

Arguments

None.

Description

Displays version numbers of the communication interface, application interface and motion interface and release/build dates associated with these. Each line of data begins with !S (as in status) as illustrated below.

 

Firmware versions of optional peripherals (prealigners, flippers/grippers) will also be reported if present and configured.

Response

!S [cookie] version<tab>{info}

Events

!E END-VERSION<tab>{host command, as received}

Examples

version

!S version COMMUNICATION-INTERFACE = 5.00-RT Jul 28 2003, 22:11:19

!S version APPLICATION-INTERFACE = V4.01 Jul 28 2003, 22:11:15

!S version MOTION-INTERFACE = V1.0

!S version CONTROLLER-ID = 001

!S version RTOS-32 = V4.01

!S version RTKERNEL = V5.34

!E END-VERSIONversion
