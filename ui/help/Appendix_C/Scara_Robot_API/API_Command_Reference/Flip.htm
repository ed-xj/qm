Flip - Rotate the end-effector flipper to a specified angle.

type: queued

Syntax

flip {abs|rel}{angle1, angle2...} {speed|fspeed} {1-100} {acc|facc} {1-100}

Arguments

abs: Flipper angles specified are absolute (relative to flipper zero). (Default)

rel: Flipper angles specified are relative to current flipper angle.

angle1, angle2...: degrees to rotate the flipper.

speed: Flip speed as a percentage of the current global speed.

fspeed: Absolute flipper speed (regardless of current setting)

acc: Flip acceleration as a percentage of the current global setting.

facc: Absolute flipper acceleration (regardless of current setting).

Description

Move the flipper to a specified angle or series of angles. Angles specified may be absolute (with respect to zero), or relative (with respect to the current flipper angle). Absolute is the default.

 

A positive value indicates clockwise rotation as viewed looking into the end-effector toward the wrist; negative values, counterclockwise. Range depends on the flipper model, but is typically -120 to 120.

 

In addition, speed and acceleration overrides may be specified. "Speed" and "Acc" arguments will apply as percentages of the current global values, while "Fspeed" and "Facc" will be applied as absolute values. For example, if the global speed setting is 50%, "Speed 50" would yield a flip speed of 25% of maximum, while "FSpeed 100" would yield the maximum flip speed.

 

Since the FLIP command does not operate the main robot motion system, "WAIT MODONE" will not wait for a FLIP command to complete. Use "WAIT FLIPPER" in that case.

Events

Note: The following event is sent when the flip starts. To know when the flip has completed, use "WAIT FLIPPER" after the flip command.

 

!E END-FLIP<tab>{host command, as received}

Examples

flip 90

>

!E END-FLIPflip 90

 

flip -90

>

!E END-FLIPflip -90

 

flip 0

>

!E END-FLIPflip 0

 

flip -90 90 -90 0 speed 100 acc 100

>

!E END-FLIPflip -90 90 -90 0 speed 100 acc 100

 

flip relative -90 90 -90 0 fspeed 100 facc 100

>

!E END-FLIPflip rel -90 90 -90 0 fspeed 100 facc 100
