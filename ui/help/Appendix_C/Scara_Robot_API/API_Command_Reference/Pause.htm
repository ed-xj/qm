Pause - Suspend robot motion for resumption later.

type: immediate

Syntax

pause

Arguments

None.

Description

Pauses robot motion. The robot will come to a smooth stop and no error condition will be raised. Motion may be restarted with the resume command, and the robot will continue to its commanded (target) position from where it was paused. If the robot is not in motion when the pause command is issued, the motion command queue will nonetheless be suspended and will not continue until the resume command is issued (exception: the behavior of pause is undefined immediately after the reset command has been issued).
