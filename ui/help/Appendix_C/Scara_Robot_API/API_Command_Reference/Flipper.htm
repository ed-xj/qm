Flipper - Command the end-effector flipper.

type: queued

Syntax

flipper {on | off | center | zero | home}

Arguments

on: turns the flipper servo on.

off: turns the flipper servo off; the flipper can be moved by hand.

center: commands the flipper to find the center reference mark.

zero: marks the current flipper position as zero degrees.

home: finds the center reference mark and then moves the flipper to zero degrees.

Description

Flipper Calibration

When a flipper is first installed, it must be taught what physical position to use as zero degrees. A default position is taught at the factory, but it is best to perform this step after the robot and flipper have been installed in the work cell and the robot has been leveled. Calibrating the flipper consists of the following steps:

  1. Make sure the robot arm is in a position where the flipper can turn freely , and issue the flipper center command. This command should take about 30 to 40 seconds to complete. It is normal for the flipper to not be level at the end of this step.

  2. Issue the flipper off command. Using a level, turn the flipper by hand until the blade is level side-to-side.

  3. Issue the flipper zero command. This marks the zero degree position and stores the encoder count in EEPROM.

  4. Issue the flipper home command. The flipper will find the center mark and move to the zero position. This command will take 30 to 40 seconds to complete.

  5. Using a level, verify that the flipper returned to the position set in step 2. If not, repeat this process.

 

Flipper Homing

If the flipper/robot is power cycled, the flipper must be initialized before use. Since the flipper zero position calibration is stored in EEPROM, the only command required to initialize the flipper after a power cycle is flipper home. Again, make sure the robot arm is in a position that allows the flipper to rotate freely before issuing this command.

Events
