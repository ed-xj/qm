Affected Commands

GETSTATION(S) will always output the SET INPUTS command, even if no inputs have been set on the station. In that case the line output would be:

SET (station) INPUTS NONE, NONE, NONE

 

MOVE will enforce the "approach" input rule before moving to the specified station. This allows input qualification on moves through VIAs.

 

PICK will enforce the "approach" rule on the move to the specified station's perch position, and the "inside" rule before moving to the inside position.

 

PLACE will enforce the same rules as PICK, but will also enforce the "release" rule before releasing the wafer.

 

The RELEASE command will enforce the "release" rule for the specified station before releasing the wafer.
