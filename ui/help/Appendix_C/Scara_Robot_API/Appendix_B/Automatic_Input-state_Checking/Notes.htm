Notes

  1. The "approach" input qualifier is only applied when moving to a station's perch from outside the station; it does not apply to all moves to the station's perch. For example, in a PICK or PLACE operation, the input would be checked only before initiating the move to the station. The input line would not be checked before moving from the inside of the station back out to the perch; this is to prevent deadlock conditions with wafer presence sensing.

  2. If the station being approached is of type "elevator", then during a PICK operation the initial move to the station is actually to the "inside index â€“1" position; in this case both the "approach" and "inside" input rules will be applied (ANDed) before the move to the station.

  3. The commands GRIP OFF and VACOFF will not consult the "release" rule for a station.

  4. The action of the input rules is effected by the command SIM WAITINPUT.
