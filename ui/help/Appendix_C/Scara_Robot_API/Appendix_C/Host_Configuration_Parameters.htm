Host Configuration Parameters

Host communication-relatived variables.

Syntax

Config {Get|Set} {Host} {Args}

Arguments

Com_Port: {TCP:5001 | COM4 | CONSOLE}

Com_Baud: {19200}

EOT: {True | False}

Examples

config get host

>

config set Com_Port = TCP:5001

config set Com_Baud = 19200

config set EOT = False

 

  1. Host Type: TCP on port 5001

config get host

>

config set Com_Port = TCP:5001

config set Com_Baud = 19200

 

  1. Host type: Serial on COM 4 at 19200 baud

config get host

>

config set Com_Port = COM4

config set Com_Baud = 19200

 

  1. Host Type: Console IO

config get host

>

config set Com_Port = Console
