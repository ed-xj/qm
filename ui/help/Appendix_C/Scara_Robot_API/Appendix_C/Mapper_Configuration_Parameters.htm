Mapper Configuration Parameters

Configure mapper-related variables.

Syntax

Config {Get|Set} Mapper {Args}

Arguments

Sim: Specifies whether to simulate the mapper.

True : Simulate mapper operation.

False: Normal operation.

Type: This specifies the type of mapper and can be any of the following types

WX43 : Hamma mapper reflective mapper.

EX43: Reflective mapper.

FX301 : Through-beam mapper.

MM : Through-beam Multimetrixs mapper.

Frame_X : X coordinate of the Mapper position on the robot.

Frame_Y : Y coordinate of the Mapper position on the robot.

Frame_Z : Z coordinate of the Mapper position on the robot.

Window_300mm : Valid index range used in wafer detection.

Window_200mm : Valid index range used in wafer detection.

Window_150mm : Valid index range used in wafer detection.

Window_Mask : Valid index range used in wafer detection.

Window_300mm_Width : Valid wafer width used in wafer detection.

Window_200mm_Width : Valid wafer width used in wafer detection.

Window_150mm_Width : Valid wafer width used in wafer detection.

Window_Mask_Width : Valid wafer width used in wafer detection.

Time_Stamp : Mapping data will be displayed with a time stamp.

Examples

To see all the Mapper related configuration variables:

config get Mapper

>

config set Mapper Sim = False

config set Mapper Type = None

config set Mapper Frame_X = 0.000

config set Mapper Frame_Y = 0.000

config set Mapper Frame_Z = 1.600

config set Mapper Window_300mm = 0.400

config set Mapper Window_200mm = 0.400

config set Mapper Window_150mm = 0.400

config set Mapper Window_Mask = 0.400

config set Mapper Window_300mm_Width = 1.750

config set Mapper Window_200mm_Width = 1.750

config set Mapper Window_150mm_Width = 1.750

config set Mapper Window_Mask_Width = 1.750

config set Mapper Time_Stamp = False
