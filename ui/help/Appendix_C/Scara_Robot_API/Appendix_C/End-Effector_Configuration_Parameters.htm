End-Effector Configuration Parameters

Configure various end-effector related variables. Note that the subsystem identifier is specific to the end-effector who's parameters are being specified. The subsystem identifier is "endeffector1" for the primary end-effector, and "endeffector2" for the secondary end-effector.

Syntax

Config {Get|Set} {Endeffector1| Endeffector2} {Args}

Arguments

Type: This specifies the type of end effector and can be any of the following types.

None : No endeffector mounted.

VacGrip : Vacuum end effector.

GripGrip : Edge gripping end effector.

RampedGrip : Edge gripping end effector.

MMGrip : Multimetrixs edge gripping end effector..

MMVacGrip : Multimetrixs double-sided vacuum. end effector.

SimGripper : Specifies whether to simulate the grip operation.

True : Simulate gripper operation.

False: Normal operation.

Sim_Grip_Detect : Specifies whether to simulate only the grip detection part of the grip command.

(When "SimGripper" is false, and "Sim_Grip_Detect" is true, the gripping operation will take place, but its success or failure will not be checked.)

True : Simulate grip detection operation.

False: Normal operation.

Frame_X : X coordinate of the endeffector center point.

Frame_Y : Y coordinate of the endeffector center point.

Frame_Z : Z coordinate of the endeffector center point.

Frame_R : Yaw in case of RJ3.

Com_Port: Specifies the serial port to which the endeffector is connected to.

Com_Baud: Specifies the baud rate for serial communication. This is currently fixed at 19200.

HWSync: To reduce latency involved due to the communication between the SBC and Endeffector, a low level hardware handshake is used.

True : Hardware handshake is enabled.

False: Hardware handshake is disabled.

Kp, Ki, Kd, Il : PID parameters for the flipping endeffector.

Torque_limit : This specifies the flipper torque limit and can be between 0 and 100. The higher the value, the higher the flipperâ€™s deservoing threshold.

R_Axis : This specifies if a rotational axis (Flipper) is present on the endeffector.

Vac_Thresh: Specifies the threshold for the endeffector vacuum. This can be between 0 and 1.

Examples

To see all the Endeffector 1 related configuration variables:

config Get Endeffector1

>

config set EndEffector1 Type = VacGrip

config set EndEffector1 SimGripper = False

config set EndEffector1 Sim_Grip_Detect = False

config set EndEffector1 Frame_X = 187.706

config set EndEffector1 Frame_Y = 0.000

config set EndEffector1 Frame_Z = 0.000

config set EndEffector1 Frame_R = 0.000

config set EndEffector1 Com_Port = UnInitialized

config set EndEffector1 Com_Baud = 19200

config set EndEffector1 HWSync = False

config set EndEffector1 Kp = 130

config set EndEffector1 Ki = 80

config set EndEffector1 Kd = 0

config set EndEffector1 Il = 100

config set EndEffector1 Torque_Limit = 100

config set EndEffector1 R_Axis = None

config set EndEffector1 Vac_Thresh = 0.25

config get Endeffector2
