Servo Configuration Parameters

Configure various servo parameters. Do not change these parameters!

Syntax

Config {Get|Set} { Servo_Parameters} {Args}

Arguments

Update_Time: Specifies how often the SBC updates robot position and velocity.

Prime_Pts: Number of points to be in the motion queue at any given time.

Prime_Out:

Queue_Max: Maximum size of the motion queue.

Queue_Min: Minimum size of the motion queue

DoneZone[{Zaxis|Shoulder|Elbow|Wrist}]: Region in which a move is considered completed, in encoder counts.

NearDoneZone[{Zaxis|Shoulder|Elbow|Wrist}]: Region in which a move is considered nearly completed, in encoder counts.

Examples

config get Servo_Parameters

>

config set Servo_Parameters Update_Time = 0.016

config set Servo_Parameters Prime_Pts = 3

config set Servo_Parameters Prime_Out = 0

config set Servo_Parameters Queue_Max = 3

config set Servo_Parameters Queue_Min = 0

config set Servo_Parameters DoneZone[Zaxis] = 0.020

config set Servo_Parameters DoneZone[Shoulder] = 0.010

config set Servo_Parameters DoneZone[Elbow] = 0.010

config set Servo_Parameters DoneZone[Wrist] = 0.010

config set Servo_Parameters NearDoneZone[Zaxis] = 0.100

config set Servo_Parameters NearDoneZone[Shoulder] = 0.020

config set Servo_Parameters NearDoneZone[Elbow] = 0.020

config set Servo_Parameters NearDoneZone[Wrist] = 0.020
