Error Event Messages (101-150)

Number 102
Message (Internal) Could not add instruction to the queue.
Applies to Operation
Recovery Reset
Action If fails again after RESET, cycle power. Check internal hub for power.

 

Number 104
Message Prealignment failed.
Applies to Align
Recovery Reset
Action Verify that a precious align has finished.

 

Number 105
Message Attempt to start prealigner while already aligning.
Applies to Align
Recovery Reset
Action Verify that a previous align has finished.

 

Number 106
Message First argument to Output command must be an integer 1-4.
Applies to Output, wait input, sim, unsim
Recovery Reset
Action Verify that the output you are attempting to use is between 1-4.

 

Number 107
Message Second argument to Output command must be ON or OFF.
Applies to Output
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 109
Message Could not connect to flipper/gripper.
Applies to Power up
Recovery Cycle power
Action If error persists, remove flipper cover and check for blinking operation light. If blinking replace RS-485 unit in controller. If not blinking, check fuses in controller and in RMU.

 

Number 110
Message Flipper error.
Applies to Operation
Recovery Reset
Action Check extended error code.

 

Number 111
Message Gripper failed to grip.
Applies to Pick
Recovery Reset
Action Verify there si a wafer and the wafer is positioned correctly.

 

Number 112
Message Gripper failed to ungrip

Recovery Reset
Action Verify that the wafer is positioned correctly.
