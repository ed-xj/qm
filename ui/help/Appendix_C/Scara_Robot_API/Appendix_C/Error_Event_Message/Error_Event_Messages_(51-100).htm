Error Event Messages (51-100)

Number 51
Message Error initializing prealigner.
Applies to Power up
Recovery Reset, initprealigners
Action Check cable from controller to prealigner for good connection.

 

Number 52
Message Prealigner error
Applies to Align, Pick, Place
Recovery Reset
Action Check cable from controller to prealigner for good connection. Consult extended error code reference.

 

Number 53
Message Turn off the teach pendant switch before executing motions.
Applies to Motion commands
Recovery Reset
Action Verify that the teach pendant has been turned off.

 

Number 55
Message Motion-command queue is full.
Applies to Motion commands
Recovery Reset
Action Send the controller fewer motion commands at a time.

 

Number 57
Message Speed parameter must be in range 1-100.
Applies to Motion commands, set speed
Recovery Reset
Action Do not set a speed value below 1 or greater than 100.

 

Number 58
Message Acc parameter must be not less than 20.
Applies to Motion commands
Recovery Reset
Action Do not set an accel value less than 20 or greater than 500.

 

Number 60
Message No r-theta solution at learn position.
Applies to Learn
Recovery Reset
Action Move joints + or - 260 degrees and retry command.

 

Number 61
Message Length of end-effector 1 is incorrect. (On TP: Menus/Setup/Frames -- frame )
Applies to Power up
Recovery Cycle power
Action Correct end-effector length and orientation using teach pendant, Menus/setup/Frames, tool frame 1, then cycle power.

 

Number 62
Message Could not read tool frame 1 from controller.
Applies to Power up
Recovery Cycle power
Action Check internal hub for power.

 

Number 63
Message Could not read link lengths from controller.
Applies to Power up
Recovery Cycle power
Action Check internal hub for power.

 

Number 64
Message Motion command failed.
Applies to Motion commands
Recovery Reset
Action Retry command. If fails again, check internal hub for power.

 

Number 65
Message Could not clear fault in controller.
Applies to Reset
Recovery Cycle power
Action Check teach pendant for error messages.

 

Number 66

Number 67

Number 68
