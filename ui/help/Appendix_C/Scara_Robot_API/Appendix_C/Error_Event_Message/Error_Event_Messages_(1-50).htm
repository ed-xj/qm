Error Event Messages (1-50)

Error numbers are not contiguous; error numbers not appearing in these table are not implemented.

 

Number 1
Message (Internal) Controller configuration, Variable_set.
Applies to Power up, configuration
Recovery Cycle power
Action Insure internal hub has power.

 

Number 2
Message (Internal) Could not connect to motion controller.
Applies to Power up, configuration
Recovery Cycle power
Action Insure internal hub has power.

 

Number 3
Message Command buffer overflow
Applies to Complete API
Recovery Reset
Action Reduce the number of commands being sent to the robot at a single time.

 

Number 4
Message First command after fault, abort or power-up must be RESET.
Applies to Complete API, power-up
Recovery Reset
Action  

 

Number 5
Message Bad command identifier.
Applies to Complete API
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 6
Message (Internal) Could not read position from controller.
Applies to Learn
Recovery Cycle power
Action Insure internal hub has power.

 

Number 8
Message Wrong number of arguments.
Applies to Complete API
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 10
Message Too many stations defined.
Applies to Learn
Recovery Reset
Action Remove any repetitive station names.

 

Number 12
Message File name required.
Applies to Execute
Recovery Reset
Action Specify a file name to be executed.

 

Number 13
Message Error writing to local file.
Applies to Save
Recovery Reset
Action "remove" unneeded station files.

 

Number 14
Message Exactly one parameter required.
Applies to Save, restore, remove, vacon
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 15
Message Error reading from file.
Applies to Execute
Recovery Reset
Action File be corrupt. Power cycle and retry command.

 

Number 16
Message Too few arguments.
Applies to Config, set, motion commands
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 18
Message Too many arguments
Applies to Complete API
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 19
Message Argument(s) must be (a) number(s).
Applies to Configuration, xyzwpr, set, flip, grip
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 20
Message Number of slots must be non-negative.
Applies to Set
Recovery Reset
Action Do not use a negative number when setting the number of slots for a station.

 

Number 21
Message Number of slots must be an integer.
Applies to Set
Recovery Reset
Action Do not use a non-integer number when setting the number of slots for a station.

 

Number 22
Message Invalid argument.
Applies to Configuration, get, set, rcp, sim, unsim, map
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 23
Message Pitch must be numeric.
Applies to Set
Recovery Reset
Action Do not use a non numeric value when setting the pitch for a station.

 

Number 24
Message Stroke must be numeric.
Applies to Set
Recovery Reset
Action Do not use a non numeric value when setting the stroke for a station.

 

Number 25
Message Align must be 0(false) or 1(true).
Applies to Set
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 26
Message Key for SET command is not a station property.
Applies to Set
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 29
Message Argument must be a positive integer.
Applies to Get, output, motion commands, sim, unsim
Recovery Reset
Action Do not use a non positive integer.

 

Number 30
Message Integer argument has a fractional part.
Applies to Get, output, motion commands, sim, unsim
Recovery Reset
Action Do not use integer with a fraction part.

 

Number 31
Message (Internal) Failed to send station database.
Applies to GetStations
Recovery Cycle power
Action Check Ethernet connection between R-J3 and SBC.

 

Number 32
Message Undefined station.
Applies to Motion commends, map, getstation
Recovery Reset
Action Verify that the station name you are using exisits in the current database.

 

Number 34
Message Blend must be in range -1 to 100.
Applies to Motion commands
Recovery Reset
Action Do not blend a motion command with an identifier less that -1 or greater than 100.

 

Number 36
Message Bad parameter in motion command.
Applies to Motion commands
Recovery Reset
Action Verify that the command you are using is syntactically correct.

 

Number 37
Message Index out of range for station.
Applies to Motion commands
Recovery Reset
Action Verify that the index you are trying to access is a valid index for the specified station.

 

Number 38
Message Attempt to seize at perch pose.
Applies to Pick
Recovery Reset
Action Move to "inside" station before issuing command.

 

Number 39
Message Attempt to seize at wrong station.
Applies to Pick
Recovery Reset
Action Move to "inside" station before issuing command.

 

Number 40
Message Could not turn on vacuum.
Applies to Pick, Place, Seize
Recovery Reset
Action Check vacuum.

 

Number 41
Message Attempt to release at perch pose
Applies to Place
Recovery Reset
Action Move to "inside" station before issuing command.

 

Number 42
Message Attempt to release at wrong station.
Applies to Place
Recovery Reset
Action Move to "inside" station before issuing command.

 

Number 43
Message Could not turn off vacuum.
Applies to Grip, place
Recovery Reset
Action Verify vacuum line integrity.

 

Number 47
Message Attempt to change index inside station.
Applies to Motion commands
Recovery Reset
Action Verify that the motion you are attempting does not involve changing your index.

 

Number 48
Message Attempt to go inside station, but not at perch.
Applies to Motion commands
Recovery Reset
Action Verify that you are a t the stations perch position.

 

Number 49
Message Attempt to leave station, but not through perch.
Applies to Motion commands
Recovery Reset
Action Verify that you are leaving the inside of the current station through the perch of the current station.

 

Number 50
Message Command is too long.
Applies to Complete API
Recovery Reset
Action Send to controller fewer commands at a time.
