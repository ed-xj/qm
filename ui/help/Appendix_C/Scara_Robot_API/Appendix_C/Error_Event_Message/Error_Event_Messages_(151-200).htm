Error Event Messages (151-200)

Number 151
Message Could not ftp station data.
Applies to Restore, save , configuration
Recovery Reset
Action "REMOVE" unneeded station files. If persists, cycle power.

 

Number 152
Message Command line is too long.
Applies to Operation
Recovery Reset
Action Shorten the length of the command you are sending to the robot.

 

Number 154
Message (Internal) Could not add I/O command to motion queue.
Applies to Motion commands, flip
Recovery Reset
Action Check internal hub power.

 

Number 155
Message Time out waiting for flipper move completion.
Applies to Motion commands, slip
Recovery Reset
Action Increase robot speed. If persists, replace flipper.

 

Number 156
Message Unable to resume flipper.
Applies to Resume
Recovery Reset / Cycle power
Action If error persists, remove flipper cover and check for blinking operation light. If blinking, replace RS-485 unit in controller. If not blinking, check fuses in controller and in RMU.

 

Number 157
Message Unable to pause flipper.
Applies to Pause
Recovery Reset / Cycle power
Action If error persists, remove flipper cover and check for blinking operation light. If blinking, replace RS-485 unit in controller. If not blinking, check fuses in controller and in RMU.

 

Number 158
Message Could not enable hardware motion-synch.
Applies to Configuration
Recovery Reset / Cycle power
Action If error persists, contact IRSI.
