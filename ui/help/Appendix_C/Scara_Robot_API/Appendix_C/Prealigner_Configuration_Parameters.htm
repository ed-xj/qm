Prealigner Configuration Parameters

Configure various prealigner-related variables. Note that the sub system identifier is specific to the prealigner who's parameters are being specified. The subsystem identifier is "prealigner1" for the primary prealigner, and "prealigner2" for the secondary prealigner.

Syntax

Config {Get|Set} {Prealigner1| Prealigner2} {Args}

Arguments

Sim: Specifies whether to simulate the prealigner.

True : Simulate prealigner operation.

False: Normal operation.

Type: This specifies the type of prealigner. It is currently set to PA800.

Com_Port: Specifies the serial port to which the prealigner is connected to.

Com_Baud: Specifies the baud rate for serial communication. This is currently set to 19200.

Wait_Place : This specifies if the prealigner should wait for the robot to move to the perch position before starting the align process.

Calc : This variable is used to specify who calculates the center of the wafer.

False: Prealigner performs the center calculation.

True: Control performs the center calculation.

Hand_Off : This variable is used describe the interface between the prealigner and the robot endeffector.

Lifter: Used when the robot has finger-type end effector. To pick or place wafer on the

prealigner, the endeffector moves in between the wafer and the prealignerâ€™s pin lifts.

Chuck: Used when the robot has a fork-type end effector. The robot picks the wafer off the chuck directly.

Align_mode:

Prealigner: The prealigner performs all alignment activities, presenting a fully aligned wafer to the robot.

Robot: The prealigner sends back offset information to the controller, allowing the robot to

effect the X-Y offset, while the preligner sets the fiduciary angle.

Theta_Adjust : This variable is used to specify the orientation of the prealigner with respect to the end effector.

Examples

To see all the Prealigner1 related configuration variables:

config get prealigner1

>

config set Prealigner1 Sim = True

config set Prealigner1 Type = PA800

config set Prealigner1 Com_Port = UnInitialized

config set Prealigner1 Com_Baud = 19200

config set Prealigner1 Wait_Place = False

config set Prealigner1 Calc = True

config set Prealigner1 Hand_Off = Chuck

config set Prealigner1 Align_Mode = Prealigner

config set Prealigner1 Theta_Adjust = 0.000
