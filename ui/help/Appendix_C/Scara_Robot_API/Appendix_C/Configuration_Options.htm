Configuration Options

This appendix describes the configuration options available through the "Config Get" and "Config Set" commands. These commands are used to set up every user-modifiable parameter related to the robot and controller. The "Config Get" command can be used to by itself to view all available configuration parameters, or in combination with a subsystem name to get configuration parameters for that subsystem alone. "Config Set" is always used with arguments for subsystem, parameter name, and value.

 

Configuration changes are not made permanent until the "Config Save" command is issued.

 

The general syntax of the configuration commands are:

Config {Set} {Category} {Args} {Value}

Config {Get} {Category} {Args}

 

Note that issuing the "Config Get" command returns a list of "Config Set" statements. This is for ease of copying and pasting statements when making variable modifications using a host terminal program.
