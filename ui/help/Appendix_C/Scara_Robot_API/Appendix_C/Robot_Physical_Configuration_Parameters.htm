Robot Physical Configuration Parameters

Configure various robot physical parameters. With the exception of ToolFrame.Length, do not change these parameters!

Syntax

Config {Get|Set} { Robot_Parameters} {Args}

Arguments

AftArm_Length: Length of the aft arm (from shoulder to elbow), center to center, in mm.

ForeArm_Length: Length of the forearm (from elbow to wrist), center to center, in mm.

ToolFrame[{Eff1|Eff2}].Length: Distance from the wrist center to payload center, in mm.

ToolFrame[{Eff1|Eff2}].Yaw:

EncPerRev[{Zaxis|Shoulder|Elbow|Wrist}]: Number of encoder counts per motor rev.

MechRatio[{Zaxis|Shoulder|Elbow|Wrist}]: Mechanical ration of motor revs to joint stroke or revs.

Enable[{Zaxis|Shoulder|Elbow|Wrist}]: Specifies whether a given axis is enabled.

Examples

config get Robot_Parameters

>

config set Robot_Parameters AftArm_Length = 187.706

config set Robot_Parameters ForeArm_Length = 187.706

config set Robot_Parameters ToolFrame[Eff1].Length = 187.706

config set Robot_Parameters ToolFrame[Eff1].Yaw = 0.000

config set Robot_Parameters ToolFrame[Eff2].Length = 187.706

config set Robot_Parameters ToolFrame[Eff2].Yaw = 0.000

config set Robot_Parameters EncPerRev[Zaxis] = 2048.000

config set Robot_Parameters EncPerRev[Shoulder] = 2048.000

config set Robot_Parameters EncPerRev[Elbow] = 2048.000

config set Robot_Parameters EncPerRev[Wrist] = 2048.000

config set Robot_Parameters MechRatio[Zaxis] = 10.560

config set Robot_Parameters MechRatio[Shoulder] = 4.800

config set Robot_Parameters MechRatio[Elbow] = 4.800

config set Robot_Parameters MechRatio[Wrist] = 4.800

config set Robot_Parameters Enable[Zaxis] = True

config set Robot_Parameters Enable[Shoulder] = True

config set Robot_Parameters Enable[Elbow] = True

config set Robot_Parameters Enable[Wrist] = True
