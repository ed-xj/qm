Jogging Configuration Parameters

Configure jogging-related variables. Do not change these parameters!

Syntax

Config {Get|Set} { Jog_Parameters} {Args}

Arguments

T1: Controls acceleration while jogging.

T2: Controls jerk while jogging.

XY.T1: Controls acceleration while jogging in XY coordinates.

XY.T2: Controls jerk while jogging in XY coordinates.

TeachPendant_Version: Specifies version of teach pendant to be used.

  • Set to 0.1 if using Beta teach pendant. (Has a "Clear/Reset" button.)

  • Set to 1.0 if using production teach pendant. (Has "Reset" button.)

Examples

config get Jog_Parameters

>

config set Jog_Parameters T1 = 0.150

config set Jog_Parameters T2 = 0.005

config set Jog_Parameters XY.T1 = 0.150

config set Jog_Parameters XY.T2 = 0.005

config set Jog_Parameters TeachPendant_Version = 1.0
