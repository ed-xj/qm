Motion Configuration Parameters

Configure various motion parameters.

Syntax

Config {Get|Set} { Motion_Parameters} {Args}

Arguments

Initial_Speed: Initial robot speed when the controller is first powered up.

Num_Axis: Number of robot axis.

Speed_Filter1: Filter parameter used while changing speed via teach pendant or via the speed command.

Speed_Filter2: Filter parameter used while changing speed via teachpendant or via the speed command.

Joint limits: Specify operating range for joint angles, vel, acc and jerk

Jlimits[{Zaxis|Shoulder|Elbow|Wrist}].Min_Pos

Jlimits[{Zaxis|Shoulder|Elbow|Wrist}].Max_Pos

Jlimits[{Zaxis|Shoulder|Elbow|Wrist}].Max_Vel

Jlimits[{Zaxis|Shoulder|Elbow|Wrist}].Max_Acc

Jlimits[{Zaxis|Shoulder|Elbow|Wrist}].Max_Jrk

Absolute limits: Specify absolute range for joint angles, velocity, acceleration, and jerk

Absolute [{Zaxis|Shoulder|Elbow|Wrist}].Min_Pos Absolute

[{Zaxis|Shoulder|Elbow|Wrist}].Max_Pos

Absolute [{Zaxis|Shoulder|Elbow|Wrist}].Max_Vel

Absolute [{Zaxis|Shoulder|Elbow|Wrist}].Max_Acc

Absolute [{Zaxis|Shoulder|Elbow|Wrist}].Max_Jrk

Cartesian limits: Specify velocity, acceleration, and jerk for Cartesian motion.

Xylimits.Max_Vel

XYLimits.Max_Acc

XYLimits.Max_Jrk

MasterPosition[{Zaxis|Shoulder|Elbow|Wrist}] : Robot joint angles when moved against the mastering fixture.

MasterCount[{Zaxis|Shoulder|Elbow|Wrist}] : Single turn residual counts when the encoders are reset.
