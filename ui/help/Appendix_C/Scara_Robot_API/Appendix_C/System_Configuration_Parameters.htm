System Configuration Parameters

Configure various system related variables.

Syntax

Config {Get|Set} System {Args}

Arguments

Log_file_verbosity: This specifies how much information is sent to the log file.

All : All events are logged.

Log_On_Error : Only the errors are logged to the file.

Retract_on_fail : This specifies if the robot should retract from a pick or place operation in case of errors.

True : Robot move to the perch position in case of errors during pick or place operation.

False: Robot stops on errors during pick or place operation.

Rtheta : Specifies the robot mode to Rtheta or SCARA.

True: Robot mode is set to Rtheta.

False: Robot mode is set to SCARA.

Safety_pick : This enables robot to check if the end effector has a wafer before proceeding to pick from a station.

True: Robot checks for presence of wafer on end effector before a pick operation.

False: Robot moves in to pick without checking if the end effector has a wafer on it.

Examples

To see all the system related configuration variables:

config get system

>

config set System Logfile_Verbosity = Log_on_error

config set System Retract_On_Fail = False

config set System RTheta = False

config set System Max_Map_Speed = 100

config set System Safety_Pick = False

To set robot mode to Rtheta:

Config set System RTheta = True

>
