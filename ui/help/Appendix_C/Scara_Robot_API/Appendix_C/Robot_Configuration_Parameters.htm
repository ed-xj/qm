Robot Configuration Parameters

Configure various robot related variables.

Syntax

Config {Get|Set} Robot {Args}

Arguments

Sim: When the robot is simulated, the controller reacts to commands from the host as if a robot was connected. Simulated motions are modeled, and take as long as real motions would take.

True : Robot is simulated.

False: Normal operation.

Sim_WaitInput: This specifies if the robot waits for user inputs be at a certain state before proceding..

True : All inputs are simulated.

False: Normal operation.

Sim_com:

True : The communication between the controller and robot is simulated.

False: Normal operation.

HWSync: To reduce latency involved due to the communication between the controllerâ€™s processor and the motion controller, a low level hardware handshake is used.

True : Hardware handshake is enabled.

False: Hardware handshake is disabled.

HWSync_Timeout: This is the amount of time (in milliseconds) to wait for a motion complete event.

Com_Port: Specifies the serial port to which the robot is connected to.

Com_Baud: Specifies the baud rate for serial communication. This is currently fixed at 57600.

Vac_Swap: The robot with vacuum option has two vacuum lines, this variable specifies if they are logically swapped.

True : Vacuum line 1 and 2 are swapped.

False: Vacuum line 1 and 2 are not swapped.

Vac_Both: This variable specifies if both the vacuum solenoids are turned on when a grip command is issued. This allows faster vacuum build time.

True : Both the Vacuum lines are turned on when a grip command or vacon command is issued.

False: Only the specified vacuum line is turned on when a grip or vacon command is issued.

Vac_Thresh: Specifies the threshold for the house vacuum. This can be between 0 and 1.

Examples

To see all the Robot related configuration variables:

config get robot

>

config set Robot Sim = False

config set Robot Sim_WaitInput = False

config set Robot Sim_Com = True

config set Robot HWSync = False

config set Robot HWSync_Timeout = 60000

config set Robot Com_Port = COM2

config set Robot Com_Baud = 57600

config set Robot Vac_Swap = False

config set Robot Vac_Both = False

config set Robot Vac_Thresh = 0.50
