Gripper Commands

GF

Returns the <u16bit> current value of the grip force sensor.

GG

Starts grip sequence. Returns G immediately if command echo is enabled, and then returns another G when the grip is completed successfully or F if it fails.

GR

Starts release sequence. Returns G immediately if command echo is enabled, and then returns R when the release is completed successfully or F if it fails.

GU

Starts pin up sequence. Returns G immediately if command echo is enabled, and then returns U when the up motion is completed successfully or F if it fails.

GD

Starts the pin down sequence. Returns G immediately if command echo is enabled, and then returns D when the down motion is completed successfully or F if it fails.

GSG

Gets the grip status. Returns one character as follows:

B – Busy gripping or releasing

G – Gripped

R – Released

E – Error

GSP

Gets the pin status. Returns one character as follows:

B – Busy gripping or releasing

U – Up

D – Down

E – Error

GT

Returns the unsigned 16-bit value stored in the variable named type. This can be used to identify the gripper.
