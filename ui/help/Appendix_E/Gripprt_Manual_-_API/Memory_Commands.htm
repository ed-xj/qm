Memory Commands

RAM contains all of the active working values. FLASH holds all the saved values. There are two blocks of FLASH memory. One holds the factory default settings, and the other holds the configuration that is automatically loaded at power up.

 

mfas Saves all settings from RAM to FLASH memory.
mfal Loads all settings from FLASH backup memory into RAM.
mfad

Loads all settings from FLASH factory Defaults into RAM.
