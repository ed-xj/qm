Position Commands (Primarily used for diagnostics)

PG0

Gets the stepper 0 motor position. Returns the <s16bit> position in motor steps.

PG1

Gets the stepper 1 motor position. Returns the <s16bit> position in motor steps.

PG2

Gets the stepper 2 motor position. Returns the <s16bit> position in motor steps.

PS0<s16bit>

Sets the stepper 0 motor position in steps.

PS1<s16bit>

Sets the stepper 1 motor position in steps.

PS2<s16bit>

Sets the stepper 2 motor position in steps.
