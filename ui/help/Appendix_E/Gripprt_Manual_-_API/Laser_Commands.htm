Laser Commands

L0

Turns the laser off.

L1

Turns the laser on.

LR

Reads the state of the laser. Returns 1 or 0.

LMC

Clears the laser map data.

LMS

Returns the number of data points <u8bit> captured by the laser mapper.

LMD

Sends the laser map data as sets of high and low transition times as <u16bit> values in units of 0.0005 seconds. So a value of 2 would equal 1 millisecond.
