Jog Commands (Primarily used for diagnostics)

JH

Halts jog motion.

JG

Starts jogging tips in the grip direction.

JR

Start jogging tips in the release direction.

JU

Starts jogging the tip(s) in the up direction.

JD

Starts jogging the tip(s) in the down direction.

JSG

Jogs tips one step in the grip direction.

JSR

Jogs tips one step in the release direction.

JSU

Jogs the tip(s) on step in the up direction.

JSD

Jogs the tip(s) on step in the down direction.
