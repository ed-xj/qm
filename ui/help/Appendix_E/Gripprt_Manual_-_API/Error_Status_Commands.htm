Error Status Commands

ESC Clears the Error Status.
ESG Returns 1 if the error status is non-zero, 0 if there are no errors.
ESF

Returns the code of the first error encountered since the error status was clear. If the return is zero, the error status is clear indicating no errors have occurred since the last ESC. Returns a two digit hexadecimal number in text format.

ESL

Returns the code of the last error encountered. (most recently encountered error) Returns a two digit hexadecimal number in text format.
ESA

Returns the number of errors in the log followed by the code of each error. Returns error codes as two digit hexadecimal numbers in text form.

EST Returns the number of errors in the log followed by a simple text tag for each error.
