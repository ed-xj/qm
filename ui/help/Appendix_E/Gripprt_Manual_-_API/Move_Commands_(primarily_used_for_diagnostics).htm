Move Commands (primarily used for diagnostics)

MPV0<s16bit><u16bit>

Starts motion of stepper 0 to the specified position in the first param using the velocity specified in the second param with a fixed acceleration of 5000.

MP0<s16bit>

Starts motion of stepper 0 to the specified position using the jog velocity.

MP1<s16bit>

Starts motion of stepper 1 to the specified position using the jog velocity.

MP2<s16bit>

Starts motion of stepper 2 to the specified position using the jog velocity.

MH0

Starts a move to the home position for stepper 0.

MH1

Starts a move to the home position for stepper 1.

MH2

Starts a move to the home position for stepper 2.
