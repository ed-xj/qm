Communication Option Commands

c

Forces the next command to use CRC16 checksums. If there is a syntax error before the next command is received, it will be executed in the globally set mode.

t

Forces the next command to operate in text mode. If there is a syntax error before the next command is received, it will be executed in the globally set mode.

b

Forces the next command to operate in binary mode. If there is a syntax error before the next command is received, it will be executed in the globally set mode.

x

Forces the next command to use hexadecimal format for its return values.

OBE

Enables the binary parameter option globally. (Text Disabled)

OBD

Disables the binary parameter option globally. Text Enabled)

OBG

Returns E if the binary parameter option is enabled or D if it is disabled.

OCE

Enables the CRC16 checksum option globally.

OCD

Disables the CRC16 checksum option globally.

OCG

Returns E if the CRC16 checksum option is enabled or D if it is disabled.

ODE

Enables the digital control option globally.

ODD

Disables the digital control option globally.

ODG

Returns E if the digital control option is enabled or D if it if disabled.

OTE

Enables the terminating character option globally.

OTD

Disables the terminating character option globally.

OTG

Returns E if the terminating character option is enabled or D if it is disabled.

OEE

Enables the command echo option globally.

OED

Disables the command echo option globally.

OEG

Returns E if the command echo option is enabled or D if it is disabled.

ORE

Enables the response CR option globally.

ORD

Disables the response CR option globally.

ORG

Returns E if the response CR option is enabled or D if it is disabled.

BAUD<u32bit>

Sets RS232 baud rate variable named baud. The return will be sent immediately at the old baud rate and communication should not be attempted at the new rate until 3 seconds after receiving the response at the old baud rate.
