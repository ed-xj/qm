Semi Automatic Mode

Semi automatic mode allows the user/operator to transfer and recover material while maintaining some safety and error checking capabilities. Through the user interface the user/operator can pick and place material, recover from errors, and test individual components. Although some safety checks are performed prior to allowing a user/operator to initiate action, proper training must be completed to ensure there is not automation or product damage. The user/operator will be capable of damaging equipment and product in this mode. Please ensure that proper training has occurred.


  1. Station and Slot - Configure the station and slot desire.

 

  1. Semi Automatic Actions -

    • Open/Close Load Ports.

    • Load/Unload Cassettes.

    • Restore Inventory Data when using SECS/GEM or MES.

    • Pick Wafer - Used to get currently selected wafer from slot.

    • Place Wafer - Used to put currently selected wafer to slot.

    • Map Cassette - Used to map currently selected cassette.

 

  1. Casette Inventory and ID Display - Displays slots that contain wafers and their current ID.
