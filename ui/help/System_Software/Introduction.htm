Manual Mode

Manual mode allows the user/operator to operate each device directly with out the constraints of the system software. This mode is used for debugging and test only. Proper training must be completed prior to operating any device in this mode. Damage to the automation, cameras, sensors and material can occur in this mode.

 

 

  1. Semi Automatic Action - Error recover and semi automatic actions.

    • Each tool will have specific error recovery and Semi Automatic Action associated with this page.

    • Ensure the proper training has occurred prior to issuing these actions.

 

  1. Send commands directly to scara robot -

    • Issue native API commands to the scara robot.

    • View the native responses.

 

  1. Testing single components of recipes -

    • Run part or complete recipes.

    • This option is used for testing and debug only.

 

  1. Load and execute MES commands (where applicable) -

    • Communicate with the current MES system.

    • Load, search and update travelers.
