Manual Operation Safety

When moving any of the robotic components under manual control, pay close attention to their respective range of motions. Use caution in selecting which coordinate system is used (described in subsequent sections, as each system has different key commands for different movements.

 

The predominant hazard to guard against during manual operation of any of the robotic components is the inadvertent striking against a person or a piece of equipment.

 

WARNING: Always wear safety goggles when operating robotic components.

 

WARNING: When working with the robotic components inside the tool, only the person exposed to the robotic components should be able to operate them. If possible, all work should be performed from outside the tool.

 

WARNING: Before operating the tool, check the location and condition of the EMERGENCY STOP switches.

 

CAUTION: Always start moving the IR‐820 robot at slow speeds (10% or less). Increase speed as you gain confidence with the IR‐820 robot's movements.
