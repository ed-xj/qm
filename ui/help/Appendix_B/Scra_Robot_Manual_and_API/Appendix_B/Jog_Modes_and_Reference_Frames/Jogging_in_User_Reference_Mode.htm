Jogging in User Reference Mode

Creating a user reference frame (See "SetFrame" in the API manual), automatically makes that frame available for jogging. Select the frame by pressing the "Jog Mode" key until the desired user reference frame is displayed.

 

Note: Frame names longer that eight characters will only have the first and last three characters of the name displayed, separated by "..".

 

Jogging in a user reference frame mode is the same as jogging in World mode, except the X and Y movements occur with respect to a user-specified reference frame, rather than the world reference frame. In addition, by activating the Position display ("Shift" + "POSN"), then selecting F5 ("User"), the current coordinates of the robot are displayed with respect to the currently selected user reference frame.

 

(See B.6 - Moving to Offsets Within a User Reference Frame for a complete discussion of user frames.)
