Jogging in Tool Mode

The IR-820 is able to move in a purely tool-relative coordinate frame (with its center of reference at the center of an imaginary wafer on the end effector), removing the need to worry about combining individual joint moves. This means the user can tell the robot to jog the end-effector forwards, backwards, left, right, and even to rotate around the wafer center, regardless of end-effector orientation.

 

To enter Tool jog mode, press the "Jog Mode" button until TOOL is displayed on the teach pendant.
