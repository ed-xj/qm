Jogging in World Mode

In the "world" reference frame, coordinates are given with respect to spindle center, with the shoulder, elbow, and wrist folded over each other in "home" position. (The home position is defined by the command "SET HOME PERCH 0, 0, 180, 0".)

 

The following diagram shows the orientation of the world reference frame. The robot is shown at the "home" position.

 

Note that rotation is performed with respect to the payload center point. Note also that in the "home" position, jogging in the +X direction is not possible.
