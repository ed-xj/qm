Jog Modes and Reference Frames

This appendix discusses the frames of reference in which the robot can be moved via the teach pendant or via API commands. When jogging the robot, these frames of reference are called "jog modes". Otherwise, they are simply called "reference frames".

 

You can also move the robot relative to some of these reference frames using API commands.

 

The five jog modes available are:

  • Joint - used to jog each joint individually.

  • R-Theta - used to emulate the motion of an R-Theta robot.

  • Tool - reference frame based on the position and orientation of a wafer on the end effector.

  • World - reference frame based on the position and orientation of the robot (this can be changed by user to reflect a user reference frame).

  • User - reference frame based on an arbitrary, user-specified displacement of the World reference frame origin.

 

Of these, Joint, World, and User are also available for moving the robot using the API.

 

You can select jog modes by repeatedly pressing the "Jog Mode" key on the teach pendant until the desired mode is displayed on the teach pendant's screen.
