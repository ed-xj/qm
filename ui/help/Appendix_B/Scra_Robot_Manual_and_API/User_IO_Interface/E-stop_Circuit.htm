E-stop Circuit

The E-stop circuit is exposed through pin 35 (E-stop In), which requires 24V to remain engaged. Pin 34 (24VDC) may be used as a source if the use of a simple E-stop switch is desired. (The user I/O "dummy plug" shorts pins 34 and 35 together.)

 

As noted above, pin 32 (E-stop Out 1) and 33(E-stop Out 2) provide for external triggering of other E-stop circuits when the controller's E-stop circuit activates. During normal operations, these two pins provide a closed circuit. When the E-stop is activated, this circuit opens.
