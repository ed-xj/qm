User I/O Connector

The user I/O connector on the back panel of the IR-820C is a DB-37F(female) connector. The user must provide a DB-37M(male) connector to carry e-stop and user I/O connections.

 

Figure 1 - EB-37F user I/O connector (pin one indicated with green triangle)

 

The user I/O connector exposes the available user inputs and outputs, as well as various voltage and ground connections. The table below shows the user I/O connector signals. See the IR-820 Application Programming Interface Manual for instructions on accessing these signals.

The following notes apply to the User I/O port, and should be considered when interfacing with the user I/O port:

  • 24VDC (pin 2, 3, 4, 5, 34, 37) is provided for user needs. Maximum load is 1A.

  • FRAME (pin 1) can be used for shielding purposes.

  • Controller has sinking digital outputs and inputs. In order to use the digital I/Os, 24VDC should be applied between pin 6 (positive) and pin 15 (negative). the user has an option of either using the controller 24V supply or 24V supply of his choice. In the case of using the controller power supply, the user should connect pin 6 (In 24VDC) to 24VDC and pin 15 (Out Common) to GND. In the case where a different power supply is used, its voltage should be in the 20VDC to 26VDC range, and it should be capable of handling a 1A load.

  • Output 1 through Output 8 (pin 16-23)are sinking open collector outputs, with a maximum current of 100mA each.

  • Input 1 through Input 8 (pin 7-14)are used by pulling them down to pin15 (Out Common) level. Input LOW - maximum 4V. Input HIGH - minimum 18V or open.

  • PW1 through PW8 (pin 24-31) are pass-through wires that allow the user to bring the signals of his choice to the robot writ. Maximum current per wire is 1A. Maximum voltage between wires is 24VDC or 24VAC.

  • Pin 32 and 33 provide dry E-stop contact. Under normal conditions the contact is closed. In case of E-stop event it is open.

  • The user should provide 24VDC on E-stop In (pin 35) in order to enable the motor power. Removing 24VDC from this pin will disable the motors.

  • The Encoder Latch Input is faster than other inputs, in order to provide accurate position latching.
