Unpacking and Installation

This section takes you through the process of unpacking and installing the IR-820 in your work cell, and powering it up for the first time. Be sure to read this entire section thoroughly prior to performing any of the work.

 

Installation comprises the following steps:

  1. Unpacking the robot.

  2. Verifying/Setting the controller voltage.

  3. Placing the robot in the work cell.

  4. Attaching the controller to the robot.

  5. Jogging the robot.

  6. Leveling the robot.

  7. Configuring the robot.
