Powering Up the Robot and Controller

Rock the controller power switch to the "ON" position. A green light will come on the front of the controller. If the controller detects any faults during startup, a yellow light will also come on.

 

Note: You must ensure that there are no wafers located on pre-aligners during power-up.

 

Note: The controller's IP address and configuration are set at the factory to customer specifications. Unless otherwise specified by label on the controller, the default IP address is 10.0.99.99, with a subnet mask of 255.255.0.0. To connect using a terminal emulation program, specify the IP address, along with a poet number 5001. See "Configuring the Robot"  for a description of how to set the IP address.
