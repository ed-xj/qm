Attaching the Robot to the Controller

  1. Connect the two power and control cables permanently attached to the controller to the robot. The sockets on the robot are not labeled, but the plugs are keyed, so you can not place a plug in the wrong socket.

  2. If the robot requires vacuum, plug the host vacuum line into the robot. The receptacle is the green ring on the bottom rear of the robot chassis.

 

Note: To remove the vacuum line push the green ring forward toward the robot. While holding the ring in the forward position, remove the vacuum line.
