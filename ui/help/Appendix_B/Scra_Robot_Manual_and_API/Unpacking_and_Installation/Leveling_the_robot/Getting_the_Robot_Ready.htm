Getting the Robot Ready

  1. With a teach pendant plugged into the robot controller, power up the controller.

  2. Raise the Z-axis so the bottom of the forearm is approximately 150-200mm (6-8") off the robot mounting plate.

  3. Remove power from the controller. The Z-axis brake will hold the robot spindle and arm assembly in position. With the power off, the robot arm segments move easily by hand.

 

WARNING: Do not allow fingers or clothing to be caught between the robot arm links or between the arm links and the end effector! Move the Robot arm with the palm of hand, with fingers extended.

 

Figure 4 - Leveling Axes
