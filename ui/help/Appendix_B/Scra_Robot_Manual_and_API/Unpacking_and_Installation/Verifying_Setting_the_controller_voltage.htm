Verifying/Setting the controller voltage

Before plugging the controller into a power source, verify that it is configured to accept the correct source Voltage. The robot controller comes from the factory configured to use a 108-120V, single phase power source. If desired, the controller may be configured easily to use 230V, single phase power.

 

The power inlet panel is located on the back of the controller. The required source voltage is indicated on the inlet panel. In the illustration below, source voltage is configured for 115V.

 

Figure 1 - Source voltage configured for 115V

 

To configure the robot controller to use a 230V power source, perform the following steps:

  1. Verify that the power cable is disconnected from the controller.

  2. Open the door covering the red voltage indicator (the fuse box cover), as shown below.

 

Figure 2 - Opening the fuse box cover

 

  1. Remove the red fuse box from the power inlet panel.

  2. Turn over the fuse box so that the desired source voltage label is right-side up, and slide it back into the power inlet panel.

  3. Close the fuse box cover.

  4. Verify that the desired source voltage is indicated on the power inlet panel.
