Configuring for a Specific End Effector

Setting up the robot to use a customer supplied and effector requires setting the controller's configuration to specify the correct end effector type and length.

 

The following instructions apply to vacuum-grip end effectors.

  1. Mount the end effector on the robot's wrist.

  2. Attach the end effector's vacuum inlet to one of the vacuum barb on the wrist. (Use whichever is most convenient. This will be checked and configured later.)

  3. Seal the remaining vacuum barb by attaching to it a short length of vacuum hose, blocked with ball bearing.

  4. Power on the robot.

  5. Verify that an active vacuum source is plugged into the back of the robot. You should hear vacuum pulling through the end effector.

  6. Establish a host connection to the controller.

  7. From the host control program, issue the command "Reset".

  8. Issue the command "config get endeffector1". This will show all the configuration options for end effector 1. (For a complete description of all the configuration options, see the API Manual.)

  9. Verify that the end effector type is set to "VacGrip". If not, issue the command "config set endeffector 1 type = vacgrip".

  10. Verify that "SimGripper" and "Sim_Grip_Detect" are set to "false". If not, issue the commands "config set endeffector1 SimGripper = false" and "config set endeffector1 Sim_Grip_Detect = false" and "config set endeffector1 Sim_Grip_Detect = false".

  11. Issue the command "grip 1 off". The vacuum should stop flowing through the end effector. If it does not. issue the command "config set robot vac_swap = true" and then "reset". Again issue the command "grip 1 off".

  12. Place a payload on the end effector and issue the command "grip1 on". The payload should now be secured to the end effector. Issue the command "grip 1 off" to release it.

  13. Issue the command "config save". This will save configuration information you have changed up to this point.

 

In order for the robot to execute linear motions correctly, the controller has to know the length of the end effector being used. This is done by first measuring the distance from the center of the robot's wrist joint to the center of the end effector's payload, then modifying a configuration variable in the controller using a host connection.

  1. Pace a test payload with centerline markings on the end effector.

  2. Measure the distance between the center of rotation of the end effector mount and the center of the payload, in millimeters.

  3. From the host control program, issue the command "config set endeffector1 frame_x = distance in mm".

  4. Issue the command "config save".
