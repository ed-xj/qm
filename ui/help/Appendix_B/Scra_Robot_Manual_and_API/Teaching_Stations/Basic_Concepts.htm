Basic Concepts

A station is simply a pose that the robot has attained, which the operator names, and saves. The station information is stored on the controller.

 

Station information includes the following:

  • Robot joint orientation for both inside and perch positions

  • Optionally, flipper orientation for both inside and perch positions.

  • Stroke distance (distance of vertical travel to pick up or drop off a wafer).

  • Pitch distance (distance between wafer slots).

  • Station mode (R-Theta or not).

  • Station type (cassette, wafer, elevator, or pre-aligner).

  • Number of index positions (the number of horizontal or vertical wafer positions).

 

The naming of convention for all station database must follow these conventions.

  • Station names must be 8 characters or less.

  • Station names can not contain spaces or punctuation.

  • Station names are not case-sensitive.

  • Station names must be alpha-numerical.
