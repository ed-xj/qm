Teaching The Inside Position

In this section you will teach the robot the positions necessary to pick or place a wafer within a wafer cassette.

  1. Place a wafer on the center of the end effector.

  2. From the host control program, issue the command "grip 1 on". This will cause the end effector to grip the wafer.

  3. On the teach pendant, pick a jog mode that is appropriate to the kind of stations you will be teaching. Generally, you should use "Tool" or "World" mode for most situations. Use "R-Theta" mode only when specifically teaching an R-Theta station.

  4. On the teaching pendant, verify that the jogging speed is below 5%.

  5. Jog the robot so the end effector and wafer are facing the cassette.

  1. Jog the robot so that the wafer is centered in the bottom cassette slot.

  1. Jog the robot to extend the wafer far enough into the cassette to guarantee the wafer is balanced correctly.

  2. Turn the teach pendant switch to the off position.

  3. In the host program, type the command "learn station_name inside", where station_name is the label you wish to apply to this station.

  4. Press "Enter" on the host keyboard to execute the command.

 

Note: The naming convention for all station is as follows:

  • Station names must be 8 characters or less.

  • Station names can not contain spaces or punctuation.

  • Station names are not case-sensitive.

  • Station names must be alpha-numerical.
