Station Touch-up

Sometimes a taught station will not be precisely where you need it to be. Station touch-up refers to making minor adjustments to the station position.

 

In this example, we will assume that the stationâ€™s inside position needs touching-up.

  1. If the inside of the station is in need of a touch up remove the wafer from index 1.

  2. Verify that the robot is in a safe position to return to the perch position.

  3. Type in the command "move station_name". This will move the robot arm to that station's perch position.

  4. Press "Enter" on the host keyboard to execute the command.

  5. Type in the command "move station_name inside". The robot will move to the inside position.

  6. Press "Enter" on the host keyboard to execute the command.

  7. Using the teach pendant, adjust the robot's position until it is satisfactory.

  8. Turn the teach pendant on-off switch to the off position.

  9. Type the command "learn station_name inside".

  10. Press "Enter" on the host keyboard to execute the command.

 

The inside position is now re-taught.

 

Note: Any time you relearn the inside position of a station it is a good idea to relearn the perch position in relation to the new inside position.

 

Note: Be sure to save your station database once you are satisfied with the positions. The re-taught positions will be lost otherwise.
