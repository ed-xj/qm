Setting Station Parameters

Now you must choose the number of slots that the stations will support. You will also need to select the pitch and stroke of the station.

 

Each station will initially default to the following settings:

Index, Type, Slots, and EFNum will be covered later in this document. From the following table, choose the pitch and stroke values appropriate to the station you just taught. You may also choose your own pitch and stroke, based on your situation.

 

  1. In the host program, type in the command "set station_name slots slot#", where station_name is the name of the station just taught, and slot# is the number of slots desired.

  2. Press "Enter" on the host keyboard to execute the command.

  3. Type in the command "set station_name pitch pitch#", where pitch# is the desired pitch you determined above.

 

Note: You should not type "mm" following the pitch number.

 

  1. Press "Enter" on the host keyboard to execute the command.
