Learning The Perch Position

  1. Turn the teach pendant switch to the on position.

  2. Jog the robot so the end-effector moves out of the FOUP to a position that is clear of any obstructions (such as a pod door opener). Be sure not to change the Z-axis position while retracting the robot.

  3. Turn the teach pendant switch into the off position.

  4. In the host program, type the command "learn station_name perch", where station_name is the name you specified when you taught the inside position.

  5. Press "Enter" on the host keyboard to execute the command.
