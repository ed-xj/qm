Testing Learned Station

Once you have taught a series of stations, you will need to verify that those stations work correctly.

 

The IR-800's control language allows you to pick or place wafers at a station by using the commands "pick" and "place". These commands automatically move the robot through the appropriate sequences of lowering the Z-height, moving into a cassette, raising the Z-height to lift the wafer, gripping the wafer, exiting the cassette, and moving to the perch position.

 

You may also command the robot to move directly to a station using the "move" command. When you execute the "move" command to move to a particular station, you will by default move to the perch position. This is to prevent inadvertently entering a cassette or tool chuck. You may move the robot to a station’s inside position, but only by moving to that station’s perch position first. If you attempt to move to an inside position without first moving to the perch, you will receive an error message.

 

The format of the "pick" command is as follows:

[cookie] pick station_name (motion_arguments)

 

(The cookie argument allows you to place a numeric identifier by the command to facilitate matching of commands with their responses. The cookie argument is optional.)

 

The format of the "place" and "move" commands is the same. See the M800 Robot Applications Programming Interface manual for a complete description of these commands and related motion arguments.

 

  1. Set your speed to 3%.

  2. Verify that a wafer is present on the end-effector.

  3. Ensure that the teach pendant and robot stop button are within reaching distance.

  4. Type in the command "place station_name index".

  5. Press "Enter" on the host keyboard to execute the command. The robot will now place the wafer at the station specified by station_name, in the slot number specified by index.

 

If something goes wrong, you can either fine tune the station position (called "touching up the station", or you can re-teach the entire station. See the section below, "Station Touch-up". If everything worked as predicted, you can attempt to pick the wafer.

 

  1. Type the command "pick station_name index".

  2. Press "Enter" on the host keyboard to execute the command. The robot will now pick up the wafer at the station specified by station_name, in the slot number specified by index. If this succeeds you can begin to increase the speed gradually.

  3. Repeat steps 4 and 5 until you are able to reach your maximum operating speed.
