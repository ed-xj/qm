Prerequisites

Prior to teach stations, you will need to have the robot installed in the work cell and powered up, with a teach pendant connected to the controller. You should be familiar with jogging the robot in its various modes (see Appendix A for a discussion of jog modes).

 

You will also need a host communication program to connect to  the controller. This may be terminal emulation program, or a custom host control program used in the control of the entire work cell. You will use this program to type in robot commands, and to receive responses from the robot controller.

 

You will also need to know the IP address of the controller in order to communicate with it. (See the previous chapter on Installation for a discussion on setting up controller IP addresses.) You will need to have an active connection between the host communication program and the controller.

 

Note: The host connects to the controller at the previously defined IP address, using port 5001.
