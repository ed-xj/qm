Manual Operation Safety

When moving the robot under manual control (i.e., via the teach pendant), pay close attention to the robot arm's range of motion. Use caution in selecting which coordinate system is used (described in subsequent sections), as each system has different key commands for different movements.

 

The predominant hazard to guard against during manual operation is the inadvertent striking of the robot arm or end effector against a person or a piece of equipment. The robot is powered by four 1/2 horsepower motors, and can cause injury or property damage if not used cautiously.

 

WARNING: Always wear safety goggles when operating the robot.

 

WARNING: When working with the robot in a work cell, only the person exposed to the robot should be able to operate it. If possible, all work should be performed from outside the work cell.

 

WARNING: Before operating the robot, check the location and condition of the EMERGENCY STOP and LIVEMAN switches.

 

CAUTION: Always start moving the robot at slow speeds (10% or less). Increase speed as you gain confidence with the robot's movements.

 

The Teach Pendant supplied by IRSI has several switches to ensure safety. The functions of each switch are as follows:

 

EMERGENCY STOP (red button on the front of the teach pendant): Pressing this button stops the robot in an emergency, irrespective of the condition of the teach pendant enable switch. Twist the button clockwise to reset it the switch.

 

PASS CONTROL button: When the teach pendant is in control of the robot, the host cannot cause the robot to move. In order for the host system to take motion control of the robot when the teach pendant is in use, the user must press this button on the teach pendant.

 

JOG button: Pressing this button enables jogging if the teach pendant has motion control. If this button is released during jogging, the robot will come to a smooth stop.
