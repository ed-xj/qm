Hardware Safety Interlocks

The IR-820 robot system does not contain any hardware safety interlocks of its own. The work cell manufacturer is responsible for providing interlocks to ensure that the robot will be disabled when the work cell open, and that power to the robot and controller will be removed if the work cell's emergency stop switch is pressed

 

The IR-820 robot system comes with two "dummy plugs" for use with the user I/O and teach pendant connectors whenever a user I/O connection or teach pendant are not connected to the controller.

 

WARNING: These dummy plugs defeat the controller e-stop circuitry and should only be used when other protective circuitry is in place which can quickly remove power to the robot. See Chapter 5(User I/O Interface) for a description of the e-stop connection circuitry associated with the user I/O connector.
