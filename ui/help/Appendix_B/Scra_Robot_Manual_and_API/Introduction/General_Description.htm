General Description

The IR-820 robot is a Selective Compliance Assembly Robot Arm (SCARA) designed primarily for use in silicon wafer fabrication facilities. The robot system comprises the Robot Mechanical Unit (RMU), the robot controller, and optional pre-aligners.

 

The RMU includes a chassis, a Z-axis spindle providing up and down motion, a forearm, an aft-arm, a wrist plate, and a choice of vacuum or electrically actuated end-effectors. An optional flipper assembly allows the end-effector to rotate as dictated by customer requirements.

 

The robot's design allows independent orientation of the fore-arm, aft-arm, and end-effector. The robot can access points not only with radial movement but also with a normal/perpendicular approach to wafer sites that are not on the robot's radial axis - it can "reach around the corner".

 

The robot may be ordered with a number of options, including arm length (forearm and aft-arm must be equal length), Z-stroke length, and a number of vacuum and/or electrical options. A variety of end effectors are available, including vacuum single-ended, vacuum double-ended, electrically actuated, and flipper equipped.

 

An IR-820C controller provides the robot's motion control.

 

Optional pre-aligners provide precise alignment of wafer payloads prior to placement on a customer's tool chuck.

 

The robot system is described in greater detail in Chapter 3(Safety) of this document.
