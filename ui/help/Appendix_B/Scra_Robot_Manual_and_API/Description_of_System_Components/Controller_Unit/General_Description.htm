General Description

Th3 IR-820C controller provides motion control for the IR-820 robot. The control unit has ports for user I/O signals, two pre-aligners, and ethernet host connections. The two serial ports may be used for host communications in lieu of ethernet. The front panel, see Figure 5, houses the controller's indicator lights. The green "Controller Power" light is illuminated whenever the controller is powered on. The blue "Robot Power" light is illuminated only when power is applied to the robot's servomotors. The yellow "Fault" light illuminates whenever the robot is in a fault state. The host interface provides access to fault details.

The controller back panel, see Figure 6, houses the interface connectors to the robot, the host controller, the optional pre-aligners, and to house power.
