Teach pendant

The teach pendant, see Figure 9, is the user interface to the robot controller. It is used to:

 

  • Jog (move) the robot under direct user control.

  • Set system variables needed for proper robot operation.

  • Clear and diagnose controller errors.

  • Run controller-based motion programs.

 

Figure 2 shows the lay out of the teach pendant.

 

The teach pendant incorporates important safety features, including the EMERGENCY STOP switch and the DEADMAN switches. In addition, the teach pendant requires that the "SHIFT" button be held down before jogging the robot, helping to ensure against inadvertent robot motion.
