Motion Issues

Symptom: Robot won’t move when commanded. Receive the error "!E ERROR 4 - move station: First command after fault, abort, master or power-up must be RESET".

 

Remedy: Issue the "reset" command.

!E ERROR 67 - CTR Servo: SERVO: Connection to servo amp lost (1)

!E ERROR 67 - CTR Servo: SERVO: Connection to servo amp lost (1)

Remedy: Turn teach pendant e-stop switch clockwise to reset. Issue the "reset" command.

Remedy: Release live-man switch completely, then squeeze to middle position. Issue the "reset" command.

Remedy: Teach pendant defective. Return to vendor for replacement.

Symptom: Received the following error message:

!E ERROR 64 - move test1 linear: Illegal config change - Motion command failed

!E ERROR 67 - CTR Servo: Illegal config change

 

Check: During linear moves (i.e. moves from perch to inside, or moves using the "linear" motion argument) the wrist may not cross over the shoulder. This change from one side of the shoulder to the other is called a configuration change.

Remedy: Change stations so configuration change is not required.

Remedy: Remove the "linear" argument from the motion command.

 

CAUTION: When first attempting the move, use low speed and be prepared to use the e-stop to stop the robot, or damage to the robot, payload, or workspace may result.

 

 

Symptom: Received the following error:

!E ERROR 47 - move test2 inside index 4: Attempt to change index inside station

 

Check: The commanded motion inside a station must remain at the same index number. Movement between high and low positions of the same index (e.g., 3 and -3) is allowed.

Remedy: Correct the motion commands to prevent moving between indices at the station’s inside position.

 

 

Symptom: Received the following error:

!E ERROR 48 - move test2 inside: Attempt to go inside station, but not at perch

 

Check: The commanded motion to an inside station must go through the corresponding perch position.

Remedy: Correct the motion commands to require a movement to the proper perch position before moving to the inside position.

 

 

Symptom: Received the following error:

!E ERROR 49 - move test1: Attempt to leave station, but not through perch

 

Check: The commanded motion from an inside position must go through the corresponding perch position.

Remedy: Correct the motion commands to require a movement to the proper perch position before continuing to the next move.

 

 

Symptom: Received the following error:

!E ERROR 146 - move test2 index 5: Attempt to stroke while entering or leaving station

 

Check: The commanded motion between and inside and perch position of a station must remain at the same Z position.

Remedy: Correct the motion commands to prevent Z motion when moving between perch and inside positions.

 

 

Symptom: Received the following error:

!E ERROR 37 - move test1 index 5: Index out of range for station

 

Check: The commanded motion referenced an index position that is outside the set range for that station. Issue the command "getstation [station name]" to check the index value for that station.

Remedy: Correct the motion commands or the station index value to allow the desired motion.
