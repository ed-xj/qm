Recovering From Popped Belts

It is possible in certain types of collisions that the IR-820 can "pop" a belt before the system detects the collision and shuts down. The "pop" is actually the belt skipping one or more teeth on one of the arm pulleys. Once this has happened, the robot position will not be correct and further collisions are likely.

 

This process describes how to determine if a belt-pop has occurred, and how to correct it if it has.
