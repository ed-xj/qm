Correcting Belt-pop

Note that this procedure requires care and attention. Careless execution can result in additional joint error, and possibly a bent end effector.

 

If the joint error is a result of a popped belt, it can usually be corrected by popping the belt in the opposite direction. If the joint error is due to a slipped joint, remastering the robot is required, and additionally the robot should be serviced to correct the source of slippage.

 

  1. With the robot in the position described in section two (above), be sure the teach pendant switch is off, and all E-Stop buttons are in the operate position. Servo the robot by either pressing the "Reset" button on the teach pendant or sending the "RESET" command from the host. Do not proceed if there are any errors that prevent the robot from servoing.

  2. If the problem arm segment is the aft- or fore-arm, hold the root end of the arm segment (the one that it pivots on) in place with one hand, and with the other hand give a firm, momentary push in the correct direction to restore alignment. It is important to hold the root side in place when attempting to move the fore-arm so that the aft-arm does not unintentionally move. The arm will be initially springy but stiffer as you press against it, and at some point you will hear a pronounced "pop". It is possible the belt has slipped more than one tooth, so continue until the arm is in alignment.

  3. If the problem is with the end-effector, hold the wrist end of the fore-arm firmly in place and grasp the end-effector near the end. If the end-effector is on a flipper, make sure it is flat so as best to handle the sideways stress. Apply a firm, momentary pressure in the correct direction to restore alignment until you here a "pop". Check the alignment and repeat as needed.

 

If you are unable to restore correct alignment with the above procedure, such that one pop moves it past the center position, then get the arm as close as possible to the correct position, and re-master the robot.

 

In almost all cases this process restores the robot to its exact previous orientation and re-teaching of stations is not required. However, when first running the robot after this process, do so at a slow speed and have the teach pendant or other robotstop switch handy while verifying correct operation.

 

If the stations are not still correct, then the robot should be re-mastered.
