Linear Motions

In some cases, joint interpolated motions are inappropriate, and may lead to interference in the work area. In these cases, the "linear" motion argument may be used to straighten the path of the robot's payload.

 

Example:

move pdo2 linear

 

Note that linear moves cannot be performed in situation that would lead to the wrist crossing over the shoulder. This will result in an "illegal configuration change" error message, since the robot is passing through a singularity.
