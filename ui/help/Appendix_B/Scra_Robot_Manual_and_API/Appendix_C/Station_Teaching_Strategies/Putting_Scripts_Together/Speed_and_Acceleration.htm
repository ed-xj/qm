Speed and Acceleration

Once the overall shape of the sequence is determined by "blend" and "linear" arguments, the final touches are made by adjusting the speed and acceleration parameters of stations as necessary.

 

Ideally, the sequence should be able to run at 100% speed at all locations, however this is not always the case. For example, running the stroke up or down moves inside a cassette at 100% speed might be too jarring, depending on the payload, in that case, the "sspeed" argument on the pick or place could be used to slow that portion of the operation to acceptable levels.

 

Note that the percentage specified is a percentage of the current speed, not maximum speed.

 

Some examples:

pick pdo1 index 5 sspeed 50 ; During the pick, stroke at 50% speed

place pdo1 index 10 ispeed 60 ; During the place, move inside at 60% speed

pick pdo3 index 1 ispeed 90 ospeed 60; Go in at 90%, leave at 60%

move via1 speed 80 ; Make the move to "via1" at 80% speed

 

The use of acceleration arguments can smooth out moves be reducing the acceleration rate. This may be desirable in applications where friction alone is holding a payload.

 

Example:

pick pdo1 acc 70 ; Accelerate at 70%
