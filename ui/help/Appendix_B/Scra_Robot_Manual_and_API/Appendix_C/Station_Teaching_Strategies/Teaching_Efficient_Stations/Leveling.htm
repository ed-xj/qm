Leveling

Start the process by leveling the workspace. Perform leveling in the following order:

  1. Begin with the robot, leveling the spindle.

  2. If the workspace includes one or more prealigners or other such intermediate staging equipment, level those next.

  3. Using the drawings from the planning steps above, jog the robot to the approximate inside station position for the prealigner, and level the end effector, using the wrist leveling mechanism, until it is level with the prealigner. If using more than one prealigner, level the second prealigner to the end effector, after adjusting the end effector to the first prealigner.

  4. Level the process tools to the end effector.

  5. Level the payload sources (e.g., FOUPs, cassettes) to the end effector.

 

In some cases, compromises may be required in the amount of leveling that is desirable as opposed to what is achievable.
