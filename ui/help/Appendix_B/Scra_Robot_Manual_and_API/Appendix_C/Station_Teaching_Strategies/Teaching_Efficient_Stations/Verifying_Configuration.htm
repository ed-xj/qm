Verifying Configuration

Before the tool, world, and user reference frames can be used, the length of the end effector must be set correctly.

 

Issue the command "config get endeffector1". Verify the value for "Frame_X" represents the distance between the wrist center and the payload center of the end effector in millimeters. If it is not correct, issue the following commands:

config set EndEffector1 Frame_X = [length in mm]

config save

If a second end effector is being used, repeat this step using "config get endeffector2".
