Asserting Control

For safety, control of the robot is exclusive: The robot may be under the control of the teach pendant, or under the control of the host, but not both at the same time.

 

To pass control to the teach pendant from the host:

  1. From the host communication program, issue the command "PASS CONTROL".

 

This releases control to the next entity to assert control. If the teach pendant is used to jog the robot after the "PASS CONTROL" command is issued, control passes to the teach pendant. If the host is used to control the robot after the "PASS CONTROL" command is issued, the host regains control.

 

To pass control from the teach pendant to the host:

  1. On the teach pendant keypad, press the "Pass Control" button.
