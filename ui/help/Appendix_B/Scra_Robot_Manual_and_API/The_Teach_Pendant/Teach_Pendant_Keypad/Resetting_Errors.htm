Resetting Errors

To reset the controller after experiencing an error while using the teach pendant, press and hold the "Shift" key and then press the "Clear/Reset" button. This will clear any existing errors, and will re-servo the robot if it has been de-servoed. If the E-stop button has been pressed, reset it by rotating it clockwise, then hit "Shift" and "Reset".

 

If the robot is de-servoed because the liveman switch has been released, re-engage the liveman switch by holding it mid-way between the fully released and fully depressed position, then hit "Shift" and "Reset".

 

If the robot is de-servoed because liveman switch has been fully depressed, re-engage the liveman switch by releasing it, then holding it mid-way between the fully released and fully depressed position. Hit "Shift" and "Reset" to re-servo.
