Menu Navigation

Menu Items may be selected in two ways:

  1. Use the arrow keys to highlight the desired menu item, then press the "Enter" button to select that item, or

  2. Use the numeric keys to highlight the desired menu item, then press "Enter" button to select that item.

 

Some displays also make of the function keys ("F1", "F2", etc.). The bottom display row is used to indicate the  active function of these keys when available.
