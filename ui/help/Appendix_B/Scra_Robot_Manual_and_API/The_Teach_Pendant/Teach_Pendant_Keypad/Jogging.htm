Jogging

The blue-background keys control jogging functions. Pressing and holding the "Jog" button enables the jogging action buttons on the right side of the keypad. Pressing the "Jog Mode" button cycles through the available jog modes. (See Appendix A, Jog Modes.)

 

The jogging action buttons have multiple labels, corresponding to the action taken in the various jog modes. Depending on the application, all buttons may not be active. For instance, if the application does not involve a fifth axis, the "Flip/Wrist2" buttons will be inactivate.

 

Following tables describe how the jogging action buttons behave in the available jog modes.

 

Joint Mode Jogging Button Action
+Z/-Z Move Z axis up/down.
+/- X/R/Shoulder Rotate shoulder counterclockwise/clockwise.
+/- Y/Theta/Elbow Rotate elbow counterclockwise/clockwise.
+/- Wrist 1 Rotate wrist counterclockwise/clockwise.
+/- Flip/Wrist 2 Rotate flipper.

 

R-Theta Mode Jogging Button Action
+Z/-Z Move Z axis up/down.
+/- X/R/Shoulder Move end effector radially.
+/- Y/Theta/Elbow Rotate arm and end effector counterclockwise/clockwise.
+/- Wrist 1 Rotate wrist counterclockwise/clockwise.
+/- Flip/Wrist 2 Flipper rotation.


World Mode Jogging Button Action
+Z/-Z Move Z axis up/down.
+/- X/R/Shoulder Tool center in the +/- X direction, with respect to World Reference Frame.
+/- Y/Theta/Elbow Tool center in the +/- Y direction, with respect to World Reference Frame.
+/- Wrist 1 Rotation about Tool Reference Frame, counterclockwise/clockwise.
+/- Flip/Wrist 2 Flipper rotation.


User Mode Jogging Button Action
+Z/-Z Move Z axis up/down.
+/- X/R/Shoulder Tool center in the +/- X direction, with respect to Selected User Frame.
+/- Y/Theta/Elbow Tool center in the +/- Y direction, with respect to Selected User Frame.
+/- Wrist 1 Rotation about Tool Reference Frame, counterclockwise/clockwise.
+/- Flip/Wrist 2 Flipper rotation.

 

Tool Mode Jogging Button Action
+Z/-Z Move Z axis up/down.
+/- X/R/Shoulder Tool center in the +/- X direction, with respect to Tool Reference Frame.
+/- Y/Theta/Elbow Tool center in the +/- X direction, with respect to Tool Reference Frame.
+/- Wrist 1 Rotation about Tool Reference Frame, counterclockwise/clockwise.
+/- Flip/Wrist 2 Flipper rotation.

 

To jog:

  1. Press the "Jog Mode" key repeatedly until the desired jog mode is shown on the teach pendant display.

  2. Press and hold the "Jog" key.

  3. Press the jogging action key associated with the desire motion.

 

Note: Jogging will stop immediately if the "Jog" button is released. To continue jogging, release the jogging action button, press and hold the "Jog" button, and continue.
