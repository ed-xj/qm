Entering Alpha-numeric Text

Some menus will allow the entry of alpha-numeric text, for example, station names. The numeric keys, and the "+/-" key are used for this purpose. Each key represents a numeral and three or four alphabetic or special characters.

 

To enter alpha-numeric text:

  1. Press the desired alpha-numeric key repeatedly until the desired letter or number appears under the cursor on the display.

  2. Press the right arrow key to move the cursor to the next position.

  3. Repeat the process until complete.

  4. Press the "Enter" key to accept the entered values.
