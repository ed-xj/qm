Appendix A

Appendix A will go over the valid set of recipe commands. Each commands is valid given certain states of operation. Please ensure you have received the proper training prior to testing changing or creating recipes.

 

COMMAND - Valid command to issue in recipe.

DEVICE - Device that the command will be sent to.

PROCESS 1 - Can the command run as process 1.

PROCESS 2 - Can the command run as process 2.

WAIT UNTIL COMPLETE - Will the scheduler wait for completion of command.

WILL JUMP - Can the command jump inside the recipe.

EXAMPLE - Recipe example of usage

DESCRIPTION - Description of command.

NOTES - Notes on command.
