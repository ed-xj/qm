CLIP_POSITION

COMMAND CLIP_POSITION
DEVICE ROBOT (APPLICATOR)
PROCESS 1 YES
PROCESS 2 NO
WAIT UNTIL COMPLETE YES
WILL JUMP YES
EXAMPLE #CLIP_POSITION#PRECMD 2,-CDMC##

 

DESCRIPTION

Returns the current position of the available clip.

 

NOTES

This command is used to pick and to place a clip from a clip drum. When placing a clip using the command #CLIP_POSITION#PRECMD 2,-CDMS##, when picking a clip use the command #CLIP_POSITION#PRECMD 2,-CDMC##.
