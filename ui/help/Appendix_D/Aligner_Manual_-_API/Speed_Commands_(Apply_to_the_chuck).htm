Speed Commands (Apply to the chuck)

There are 4 selectable speeds that apply to the move, jog, and stationing commands. These 4 speeds are numbered 0-3 with 0 defaulting to the slowest speed and 3 the fastest. Each of these speeds contain an acceleration and a velocity. Once a speed has been selected, it remains selected until another speed is selected. The values for the acceleration and velocity may be edited in RAM and stored and recalled from flash. At startup, the RAM values are loaded from flash and speed 1 is selected. The jogging commands operate at speed 0. The units for the velocities are interpolator units per sample. The acceleration values are in interpolator units per sample squared. See the trajectory commands section for unit conversions.

 

SS#

Selects the speed number #. The range of the number # is 0-3 where 0 defaults to the slowest and 3 to the fastest.
SVW#<u32bit> Writes Speed # Velocity.
SAW#<u32bit> Writes Speed # Acceleration.
SVR# Reads Speed # Velocity Returns <u32bit> value.
SAR# Reads Speed # Acceleration. Returns <u32bit> value.
