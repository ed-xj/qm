Vacuum Commands

VAC0

Turns chuck vacuum off.

VAC1

Turns chuck vacuum on (to hold wafer).

VACR

Reports the state of the chuck vacuum value. Return 1 if on and 0 if off.

VACS

Reports the state of the chuck vacuum sensor. Returns 1 if the vacuum sensor reads a value above the variable vac_min indicating the wafer is held and 0 if not.
