Servo Commands (Apply to the chuck)

SVE

Enables servo control of the slide motor. This must be enabled for the slide to perform moves and maintain its position. The error status must be clear prior to executing this command.

SVD

Disables the servo allowing for manual positioning of the slide. This may be useful in manually teaching a station position.

SVG

Gets servo status. Returns E if enabled and D if disabled.
