Stationing Commands (Apply to the chuck)

There are 10 station positions, numbered 0 through 3, plus 3 special stations, that may be taught or edited in RAM, and saved to non‚Äêvolatile flash memory. On startup, the flash values are loaded. See the memory commands to see how to store all current settings into flash memory. The three special stations are Z, P, and N. Z is the user zero offset that modifies the zero position. This value should normally be zero. At the zero position, the aligner is near the negative end of its full motion. P is the positive move limit position. And N is the negative move limit position.

 

NT# Teaches the current position for station position #.
NRM# Returns the station position in millidegrees <s32bit>.
NRC# Returns the station position in counts <s32bit>.
NWM#<s32bit> Writes the station position in millidegrees.
NMC#<s32bit> Writes the station position in counts.
NM#

Moves at the currently selected speed to the position in RAM for station number #. If the slide is already in motion when this command is called, the current motion will be stopped and the command will fail.
