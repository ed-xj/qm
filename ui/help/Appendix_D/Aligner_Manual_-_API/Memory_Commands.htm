Memory Commands

Settings for the following commands include all the speeds, station positions, servo parameters, and communication options.

 

RAM contains all of the active working values. FLASH holds all the saved values. There are two blocks of FLASH memory. One holds the factory default settings, and the other holds the configuration that is automatically loaded at powerup.

 

mfas Saves all settings from RAM to FLASH memory.
mfal Loads all settings from FLASH backup memory into RAM.
mfad

Loads all settings from FLASH factory Defaults into RAM. After using this command on an aligner, the sensor will need to be re‚Äêcalibrated with the C.

 

The following commands are used by the register browser of the QUARTET MECHANICS GripUtil windows application and are not intended for use by a normal host or robot. They allow reading and writing memory locations and registers directly.

 

mr Reads a byte from RAM.
mw Writes a byte to RAM.
mfr Reads a byte from FLASH memory.
