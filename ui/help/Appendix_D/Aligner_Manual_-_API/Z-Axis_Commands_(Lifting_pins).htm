Z-Axis Commands (Lifting pins)

Note that the Z-Axis uses an open-loop micro-stepper and position values are in microsteps increasing in the upward direction.

 

ZE Enables the z‐axis and energizes the motor’s windings.
ZD Disables the z‐axis.
ZHM

Moves the lifting pins to the home flag near the bottom of their range of motion and sets the position to zero.
ZHS Returns 1 or 0 to indicate the current state of the Z‐axis home sensor.
ZMU

Turns off the vacuum to the chuck, homes the axis if needed, and moves the lifting pins to the position set by the variable pins_up_psn used for wafer transfer with a robot.
ZMD

Homes the axis if needed and moves the lifting pins to the position set by the variable pins_dn_psn which should be just below the wafer but not touching it while the wafer rests on the chuck.
ZMP<s16bit>,
        <u16bit>,
        <u16bit>

Starts move to the position specified in the first parameter at the velocity of the second parameter and acceleration of the third parameter.
ZRS

Reads the state of the Z‐axis and returns one of the following:

H if the axis is not referenced to home.

U if the axis is in the Up position.

D if the axis is in the Down position.

P if the axis is referenced to home but not in the up or down position.
ZP Returns the <s16bit> value of the Z‐axis position.
ZZ Sets the current position to zero and clears the homed state.
ZS Stops motion of the Z-Axis.
UWC

Unloads a wafer from the chuck by releasing the vacuum hold and raising the wafer off the chuck with the lifting pins.
LWC

Loads a wafer to the chuck by lowering the wafer onto the chuck with the lifting pins and taking hold of the wafer with the vacuum on.
