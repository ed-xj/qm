Move Commands (Apply to the chuck)

H

Halts motion (same as TH).

MAM<32bit>

Starts a move to the absolute position in millidegrees at the currently selected speed. Must not already be in motion.

MRM<32bit>

Starts a move to the relative position in millidegrees at the currently selected speed.

MAC<32bit>

Starts a move to the absolute position in counts at the currently selected speed. Must not already be in motion.

MRC<32bit>

Starts a move to the relative position in counts at the currently selected speed.

MIL

Causes the next move command to ignore the user position limits.

MD

Return 1 if the move is done and 0 if the last move trajectory is not complete.
