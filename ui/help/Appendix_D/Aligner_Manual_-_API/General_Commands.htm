General Commands

V

Return version code. A2.4

VV

Returns a single character revision letter.

VBD

Returns the build date of the firmware such as Nov 11 2009.

AD#

Address aligner # (where # is 0 or 1). All commands refer to the last addressed aligner in a multi-aligner system. At startup, aligner 0 is addressed. Systems with only one aligner should not use this command.
