Position Commands (Apply to the chuck)

Positions are maintained and operated upon internally in encoder counts and interpolator counts. Some commands allow the host to use micron or millidegree units instead of counts, but internally the values are just converted to and from counts. There are 8388.6 interpolator counts per micron and 128 encoder counts per millimeter. Absolute moves issued while the slide is already in motion will cause the current motion to stop and return an error.

 

PMR Returns the <s32bit> command position in millidegrees.
PCR Returns the <s32 bit> command position in counts.
PCA Returns the <s32bit> actual position in counts.
PCE Returns the <s16bit> current position servo error in counts.
PCW<s32bit>

Writes the current position in counts to the aligner.

(Shouldn’t need this command for normal operation.)
PMW<s32bit>

Writes the current position in millidegrees to the aligner.

(Shouldn’t need this command for normal operation.)

 

For internal use only:

PLP Learn positive hardstop.
PLN Learns negative hardstop.
PZ Set current position to zero - Allows for manual zeroing of the slide position.
