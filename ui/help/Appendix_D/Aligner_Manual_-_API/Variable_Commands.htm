Variable Commands

VR<name>

Reads the value of the variable named in the argument <name> from RAM. The return format depends on the variable read.
VW<name><val>

Writes the value specified in the second argument <val> to the variable named in the first argument <name> to RAM. The argument value depends on the variable being set.
VL

Outputs a list of all the variables available and their current values. This command is intended for manual use on a terminal for diagnostics. One line is output for each variable and uses the syntax of the VW command to allow simple cut and paste operations for setting new values.

 

The following variables are available:

mP <s32bit>   measured position of the chuck angle in encoder counts (72000 per rev)
mE <s32bit> measured servo error in encoder counts
pwm <u16bit> for internal use only
comm_cnfg <u16bit>* holds the communication configuration setting bits
sys_cnfg <u16bit>* holds the system configuration setting bits
baud <u32bit>* baud rate (will be used at startup if saved to flash)
cnts_per_rot <u32bit>* chuck resolution (do not change)
ks <u16bit>* scale factor for all servo gains
kp <u16bit>* proportional servo gain
ki <u16bit>* integral servo gain
ki_m <u16bit>* integral servo gain while in motion
kd <u16bit>* derivative servo gain
err_lim <u16bit>* servo error limit
isum_lim <u32bit>* integrator limit
isum_lim_m <u32bit>* integrator limit while in motion
cal_empty <u16bit>* alignment sensor calibration value with sensor completely unblocked
cal_blocked <u16bit>* alignment sensor calibration value with sensor completely blocked
vac_min <u16bit>* minimum vacuum level in ADC counts to indicate a gripped wafer
fine_calc <u16bit>* setting for align algorithms
fid_width <s32bit> diagnostic width of fiducial in millidegrees
orient_ang <s32bit>* angle of fiducial orientation after align in millidegrees
max_notch <s32bit>* max allowed value of measured width of notch in millidegrees
max_flat <s32bit>* max allowed value of measured width of flat in millidegrees
min_flat <s32bit>* min allowed value of measured width of flat in millidegrees
z_homed <u8bit> diagnostic
pins_up_psn <s16bit>* pins up position of z‐axis in stepper counts from home
pins_dn_psn <s16bit>* pins down position of z‐axis in stepper counts from home
pins_up_vel <u16bit>* velocity for pins up move of z‐axis
pins_dn_vel <u16bit>* velocity for pins down move of z‐axis
pins_up_acc <u16bit>* acceleration for pins up move of z‐axis
pins_dn_acc <u16bit>* acceleration for pins down move of z‐axis
rel_del_ms <u16bit>* millisecond delay before turning off the vacuum chuck and raising pins

 

* Using the memory commands below, all the variable values marked with an asterisk above may be transferred to and from FLASH.
