Laser Edge Sensor Commands

LPE Enables power to the laser sensor.
LPD Disables power to the laser sensor.
LE Enables the sensor.
LD Disables the sensor.
LR Returns the edge sensor reading in <u16bit> format.
LCE

Calibrates the edge sensor by setting the variable cal_empty. The sensor should be completely un-blocked when calling this function. The command will also return the value in the <u16bit> format. After calibrating both the blocked and empty variables, they should be saved to flash with the mfas command.
LCB

Calibrates the edge sensor by setting the variable cal_blocked. The sensor should be completely blocked when calling this function. The command will also return the value in the <u16bit> format. After calibrating both the blocked and empty variables, they should be saved to flash with the mfas command.
